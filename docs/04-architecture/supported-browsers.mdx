Next.js は**最新ブラウザ**を設定不要でサポートしています。

- Chrome 64+
- Edge 79+
- Firefox 67+
- Opera 51+
- Safari 12+

## Browserslist

特定のブラウザや機能をターゲットにしたい場合、Next.js は `package.json` ファイルで [Browserslist](https://browsersl.ist) の設定をサポートしています。Next.js はデフォルトで以下のBrowserslistの設定を使用します：

```json filename="package.json"
{
  "browserslist": [
    "chrome 64",
    "edge 79",
    "firefox 67",
    "opera 51",
    "safari 12"
  ]
}
```

## Polyfills

[広く使われている polyfills](https://github.com/vercel/next.js/blob/canary/packages/next-polyfill-nomodule/src/index.js) を注入します； 具体的には：

- [**fetch()**](https://developer.mozilla.org/docs/Web/API/Fetch_API) — 置き換え：`whatwg-fetch` と `unfetch`
- [**URL**](https://developer.mozilla.org/docs/Web/API/URL) — 置き換え：[`url` パッケージ (Node.js API)](https://nodejs.org/api/url.html)
- [**Object.assign()**](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object/assign) — 置き換え：`object-assign`、`object.assign`、および `core-js/object/assign`

依存関係にこれらの polyfills が含まれている場合、それらは重複を避けるために本番環境ビルドから自動的に除去されます。

さらに、バンドルサイズを減らすために、Next.js はこれらの polyfills を必要とするブラウザのみに読み込みます。グローバルに Web トラフィックの大半がこれらの polyfills をダウンロードしないことになります。

### カスタム Polyfills

ご自身のコードや外部の npm 依存関係がターゲットブラウザ（例：IE 11）がサポートしていない機能を要求する場合は、自分で polyfills を追加する必要があります；

この場合、必要な**特定の polyfill** を [Custom `<App>`](/docs/pages/building-your-application/routing/custom-app) または個々のコンポーネントでトップレベルのインポートとして追加すべきです；

## JavaScript 言語機能

Next.js は最新のJavaScript機能をすぐに使用できるようにしています。 [ES6 features](https://github.com/lukehoban/es6features)に加え、Next.js は次のものもサポートしています：

- [Async/await](https://github.com/tc39/ecmascript-asyncawait) (ES2017)
- [オブジェクト Rest/Spread プロパティ](https://github.com/tc39/proposal-object-rest-spread) (ES2018)
- [動的 `import()`](https://github.com/tc39/proposal-dynamic-import) (ES2020)
- [Optional Chaining](https://github.com/tc39/proposal-optional-chaining) (ES2020)
- [Nullish Coalescing](https://github.com/tc39/proposal-nullish-coalescing) (ES2020)
- [クラスフィールド](https://github.com/tc39/proposal-class-fields) および [静的プロパティ](https://github.com/tc39/proposal-static-class-features) (ES2022)
- その他
