---
title: 'serverExternalPackages'
description: 'bundlePagesRouterDependenciesによる依存関係のバンドリングから特定の依存関係を除外します。'
---

依存関係の自動バンドリングから特定の依存関係を除外するには、[`bundlePagesRouterDependencies`](https://nextjs.org/docs/canary/pages/api-reference/config/next-config-js/bundlePagesRouterDependencies)オプションを使います。

これらのページは、ネイティブのNode.js `require`を使用して依存関係を解決します。

```js title="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```

Next.jsは現在、互換性が進行中の[一般的なパッケージの短いリスト](https://github.com/vercel/next.js/blob/canary/packages/next/src/lib/server-external-packages.json)を含んでおり、自動的に除外されています：
