**unauthorized** ファイルは、認証中に [`unauthorized`](/docs/app/api-reference/functions/unauthorized) 関数が呼び出されたときにUIをレンダリングするために使用されます。UIをカスタマイズできることに加えて、Next.js は `401` ステータスコードを返します。

<Tabs>
<TabItem value="tsx" label="TypeScript">

```tsx title="app/unauthorized.tsx" switcher
import Login from '@/app/components/Login'

// 認証されていないページを表示するコンポーネント
export default function Unauthorized() {
  return (
    <main>
      <h1>401 - Unauthorized</h1>
      <p>Please log in to access this page.</p>
      <Login />
    </main>
  )
}
```

</TabItem>

## リファレンス {#reference}

### Props {#props}

`unauthorized.js` コンポーネントは、props を受け取りません。

## 例 {#examples}

### 認証されていないユーザーにログインUIを表示する {#displaying-login-ui-to-unauthenticated-users}

[`unauthorized`](/docs/app/api-reference/functions/unauthorized) 関数を使用して、ログインUI付きの `unauthorized.js` ファイルをレンダリングできます。

<Tabs>
<TabItem value="tsx" label="TypeScript">

```tsx title="app/dashboard/page.tsx" switcher
import { verifySession } from '@/app/lib/dal'
import { unauthorized } from 'next/server'

// ダッシュボードページを表示する非同期関数
export default async function DashboardPage() {
  const session = await verifySession()

  // セッションがない場合、認可されていないと見なします
  if (!session) {
    unauthorized()
  }

  return <div>Dashboard</div>
}
```

</TabItem>
