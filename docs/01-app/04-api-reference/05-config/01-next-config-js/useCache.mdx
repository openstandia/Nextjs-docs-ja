`useCache`フラグは、Next.jsの実験的な機能で、[`dynamicIO`](/docs/app/api-reference/config/next-config-js/dynamicIO)とは独立して[`use cache`ディレクティブ](/docs/app/api-reference/directives/use-cache)を使用できるようにします。これを有効にすると、`dynamicIO`がオフになっていても、アプリケーションで`use cache`を使用できます。

## 使用法 {#usage}

`useCache`フラグを有効にするには、`next.config.ts`ファイルの`experimental`セクションで`true`に設定します：

```ts title="next.config.ts"
import type { NextConfig } from 'next'

// NextConfig型をインポートします

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

// experimentalセクションでuseCacheをtrueに設定します

export default nextConfig
```

`useCache`が有効になっていると、次のキャッシュ関数と設定を使用できます：

- [`use cache`ディレクティブ](/docs/app/api-reference/directives/use-cache)
- `use cache`と共に使用する[`cacheLife`関数](/docs/app/api-reference/config/next-config-js/cacheLife)
- [`cacheTag`関数](/docs/app/api-reference/functions/cacheTag)
