## 非サポートAPI {#unsupported-apis}

Edge Runtime にはいくつか制限があり、以下を含みます：

- ネイティブのNode.js APIは**サポートされていません**。たとえば、ファイルシステムの読み書きができません；
- ES Modulesを実装し、ネイティブのNode.js APIを使用しない限り、`node_modules`を使用することができます；
- 直接`require`を呼び出すことは**許可されていません**。代わりにES Modulesを使用してください；

以下のJavaScript言語機能は無効であり、**動作しません：**

| API                                                                                                                             | 説明                                                                            |
| ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| [`eval`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/eval)                                       | 文字列として表現されたJavaScriptコードを評価します                              |
| [`new Function(evalString)`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Function)               | 提供されたコードを引数として新しい関数を作成します                              |
| [`WebAssembly.compile`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/WebAssembly/compile)         | バッファーソースからWebAssemblyモジュールをコンパイルします                     |
| [`WebAssembly.instantiate`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/WebAssembly/instantiate) | バッファーソースからWebAssemblyモジュールをコンパイルおよびインスタンス化します |

稀に、あなたのコードが実行時に到達できない、またはツリーシェークによって削除できない動的コード評価文を含んでいる（またはインポートされている）ことがあります。
ミドルウェア構成を使用して特定のファイルでチェックを緩和することができます：

```javascript title="middleware.ts"
export const config = {
  unstable_allowDynamic: [
    // 単一ファイルを許可します
    '/lib/utilities.js',
    // globを使用して、サードパーティのfunction-bindモジュール内のすべてを許可します
    '**/node_modules/function-bind/**',
  ],
}
```

`unstable_allowDynamic`は[glob](https://github.com/micromatch/micromatch#matching-features)またはグロブの配列で、特定のファイルに対して動的コード評価を無視します。グロブはアプリケーションrootフォルダに対する相対パスです。

これらの文がEdgeで実行される場合、*ランタイムエラーをスローし、実行時エラーが発生する*ので注意してください。
