## 非対応の API {#unsupported-apis}

Edge Runtime にはいくつかの制限があり、それには以下が含まれます：

- ネイティブの Node.js API は**サポートされていません**。たとえば、ファイルシステムの読み書きをすることはできません。
- `node_modules` は _使用可能です_ 、ただし ES モジュールを実装しており、ネイティブの Node.js API を使用しない場合に限ります。
- `require` を直接呼び出すことは**できません**。代わりに ES モジュールを使用してください。

次の JavaScript 言語機能は無効であり、**機能しません：**

| API                                                                                                                             | 説明                                                                            |
| ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| [`eval`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/eval)                                       | 文字列で表現された JavaScript コードを評価します                                |
| [`new Function(evalString)`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Function)               | 引数として提供されたコードで新しい関数を作成します                              |
| [`WebAssembly.compile`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/WebAssembly/compile)         | バッファソースから WebAssembly モジュールをコンパイルします                     |
| [`WebAssembly.instantiate`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/WebAssembly/instantiate) | バッファソースから WebAssembly モジュールをコンパイルおよびインスタンス化します |

まれに、コードに含まれる（またはインポートされる）動的コード評価文が実行時に到達できず、ツリーシェイクによって削除できないことがあります。
その場合、ミドルウェアの設定で特定のファイルに対するチェックを緩和できます：

```javascript title="middleware.ts"
export const config = {
  unstable_allowDynamic: [
    // 単一のファイルを許可
    '/lib/utilities.js',
    // function-bind サードパーティモジュール内のすべてを許可するためにグロブを使用
    '**/node_modules/function-bind/**',
  ],
}
```

`unstable_allowDynamic` は [glob](https://github.com/micromatch/micromatch#matching-features) であり、またはグロブの配列であり、特定のファイルの動的コード評価を無視します。グロブはアプリケーションの root フォルダからの相対パスです。

これらのステートメントが Edge で実行されると、*それがスローされ、実行時エラーを引き起こす*ことを警告します。
