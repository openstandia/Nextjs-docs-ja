---
title: 'typedRoutes'
description: '静的に型づけされたリンクのための実験的サポートを有効にします。'
version: 'experimental'
---

[静的に型づけされたリンク](/docs/app/api-reference/config/typescript#statically-typed-links)のための実験的サポートです。この機能を利用するには、app routerとともにプロジェクトでTypeScriptを使用する必要があります。

```js title="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
