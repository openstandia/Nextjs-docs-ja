Next.jsはデフォルトで、各ページの[etag](https://en.wikipedia.org/wiki/HTTP_ETag)を生成します。キャッシュ戦略によっては、HTMLページのetag生成を無効にしたい場合があります。

`next.config.js`を開いて、`generateEtags`オプションを無効にします：

```js title="next.config.js"
module.exports = {
  generateEtags: false,
}
```
