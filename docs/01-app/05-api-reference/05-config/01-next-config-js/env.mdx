JavaScriptバンドルに環境変数を追加するには、`next.config.js`を開き、`env`設定を追加します：

```js title="next.config.js"
module.exports = {
  env: {
    customKey: 'my-value',
  },
}
```

これで、コード内で`process.env.customKey`にアクセスできます。例えば：

```jsx
function Page() {
  return <h1>The value of customKey is: {process.env.customKey}</h1>
}

export default Page
```

Next.jsはビルド時に`process.env.customKey`を`'my-value'`に置き換えます。webpackの[DefinePlugin](https://webpack.js.org/plugins/define-plugin/)の性質上、`process.env`変数を分割代入しようとしても機能しません。

例えば、次の行は：

```jsx
return <h1>The value of customKey is: {process.env.customKey}</h1>
```

最終的には次のようになります：

```jsx
return <h1>The value of customKey is: {'my-value'}</h1>
```
