`viewTransition`は、Reactにおける新しい実験的な[View Transitions API](https://developer.mozilla.org/en-US/docs/Web/API/View_Transition_API)を有効にするための実験的なフラグです。このAPIを使用すると、ネイティブのView TransitionsブラウザAPIを活用して、UI状態間のシームレスな遷移を作成できます。

この機能を有効にするには、`next.config.js`ファイルで`viewTransition`プロパティを`true`に設定する必要があります。

```js title="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    viewTransition: true,
  },
}

module.exports = nextConfig
```

> 重要なお知らせ: この機能はNext.jsチームによって開発または保守されているものではなく、Reactチームによる実験的なAPIです。まだ**初期段階**であり、**本番環境での使用は推奨されません**。実装はまだ進行中であり、将来のReactリリースでその動作が変更される可能性があります。
> この機能を有効にするには、APIの実験的な性質を理解する必要があります。その動作を完全に把握するには、[Reactのプルリクエスト](https://github.com/facebook/react/pull/31975)と関連する議論を参照してください。

## 使用方法 {#usage}

一度有効にすると、アプリケーション内でReactから`ViewTransition`コンポーネントをインポートできます：

```jsx
import { unstable_ViewTransition as ViewTransition } from 'react'
```

しかし、現在のところドキュメントや例は限られており、これがどのように機能するかを理解するためには、Reactのソースコードや議論を直接参照する必要があります。

### ライブデモ {#live-demo}

この機能が実際にどのように動作するかを確認するには、[Next.js View Transition Demo](https://view-transition-example.vercel.app)をご覧ください。

このAPIが進化するにつれて、ドキュメントを更新し、より多くの例を共有する予定です。しかし、現時点では、この機能を本番環境で使用することは強くお勧めしません。
