> **Good to know**: Next.js 13.4以降、このオプションは**不要**になりました。App Routerは現在安定しています。

App Router（[`app` ディレクトリ](/docs/app/building-your-application/routing)）は、[layouts](/docs/app/building-your-application/routing/layouts-and-templates)、[Server Components](/docs/app/building-your-application/rendering/server-components)、[streaming](/docs/app/building-your-application/routing/loading-ui-and-streaming)、および[co-located data fetching](/docs/app/building-your-application/data-fetching)のサポートを可能にします。

`app` ディレクトリを使用すると、自動的に[React Strict Mode](https://react.dev/reference/react/StrictMode)が有効になります。[`app` を段階的に採用する方法](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app)を学びましょう。
