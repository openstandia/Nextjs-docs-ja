---
title: 'revalidateTag'
description: 'revalidateTag 関数のAPIリファレンス。'
---

`revalidateTag` は、特定のキャッシュタグに対して、[キャッシュされたデータ](/docs/app/deep-dive/caching)をオンデマンドでパージすることができます。

> **Good to know**:
>
> - `revalidateTag` は、パスが次に訪問されたときにのみキャッシュを無効化します。つまり、`revalidateTag` を dynamic route segment で呼び出しても、すぐに多くの再検証が一度にトリガーされるわけではありません。無効化は、パスが次に訪問されたときにのみ発生します。

## パラメータ {#parameters}

```tsx
revalidateTag(tag: string): void;
```

- `tag`: 再検証したいデータに関連付けられたキャッシュタグを表す文字列。256文字以下である必要があります。この値は大文字と小文字を区別します。

`fetch` にタグを追加する方法は次のとおりです：

```tsx
fetch(url, { next: { tags: [...] } });
```

## 戻り値 {#returns}

`revalidateTag` は値を返しません。

<TabItem value="ts" label="TypeScript">

```ts title="app/actions.ts" switcher
'use server'

// revalidateTag を next/cache からインポート
import { revalidateTag } from 'next/cache'

// submit 関数を非同期で定義
export default async function submit() {
  // addPost 関数を待機
  await addPost()
  // 'posts' タグを再検証
  revalidateTag('posts')
}
```

### Route Handler {#route-handler}

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="app/api/revalidate/route.ts" switcher
// NextRequest 型を next/server からインポート
import type { NextRequest } from 'next/server'
// revalidateTag を next/cache からインポート
import { revalidateTag } from 'next/cache'

// GET 関数を非同期で定義
export async function GET(request: NextRequest) {
  // リクエストから 'tag' パラメータを取得
  const tag = request.nextUrl.searchParams.get('tag')
  // 取得したタグを再検証
  revalidateTag(tag)
  // JSON レスポンスを返す
  return Response.json({ revalidated: true, now: Date.now() })
}
```
