title: 'revalidateTag'
description: 'revalidateTag 関数のAPIリファレンス'
---

`revalidateTag` は、特定のキャッシュタグに対する[キャッシュされたデータ](/docs/app/building-your-application/caching)をオンデマンドで消去できるようにします。

> **Good to know**:
>
> - `revalidateTag` は[Node.js and Edge runtimes](/docs/app/building-your-application/rendering/edge-and-nodejs-runtimes)の両方で利用可能です。
> - `revalidateTag` は、パスが次に訪問されたときにのみキャッシュを無効化します。つまり、`revalidateTag` を dynamic route segment で呼び出しても、すぐに多くの再検証が一度に始まるわけではありません。無効化は、パスが次に訪問されたときにのみ発生します。

## パラメータ {#parameters}

```tsx
revalidateTag(tag: string): void;
```

- `tag`: 再検証したいデータに関連づけられたキャッシュタグを表す文字列です。256文字以下である必要があります。この値は大文字と小文字が区別されます。

`fetch`にタグを追加するには、次のようにします：

```tsx
fetch(url, { next: { tags: [...] } });
```

## 戻り値 {#returns}

`revalidateTag` は値を返しません。

export async function GET(request: NextRequest) {

  return Response.json({ revalidated: true, now: Date.now() })
}
