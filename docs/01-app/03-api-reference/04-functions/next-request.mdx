NextRequest は [Web Request API](https://developer.mozilla.org/docs/Web/API/Request) を拡張し、利便性の高い追加メソッドを提供します。

## `cookies` {#cookies}

リクエストの[`Set-Cookie`](https://developer.mozilla.org/docs/Web/HTTP/Headers/Set-Cookie)ヘッダーを読み取るか変更します。

### `set(name, value)` {#set-name-value}

名前を指定して、リクエストに対して指定された値のクッキーを設定します。

```ts
// 受信リクエスト /home
// バナーを非表示にするクッキーを設定
// リクエストには `Set-Cookie:show-banner=false;path=/home` のヘッダーが付きます
request.cookies.set('show-banner', 'false')
```

### `get(name)` {#get-name}

クッキーの名前を指定して、クッキーの値を返します。クッキーが見つからない場合は、`undefined`が返されます。複数のクッキーが見つかった場合、最初のものが返されます。

```ts
// 受信リクエスト /home
// { name: 'show-banner', value: 'false', Path: '/home' }
request.cookies.get('show-banner')
```

### `getAll()` {#getall}

クッキーの名前を指定して、クッキーの値を返します。名前が指定されない場合、リクエストにあるすべてのクッキーを返します。

```ts
// 受信リクエスト /home
// [
//   { name: 'experiments', value: 'new-pricing-page', Path: '/home' },
//   { name: 'experiments', value: 'winter-launch', Path: '/home' },
// ]
request.cookies.getAll('experiments')
// または、リクエスト内のすべてのクッキーを取得する
request.cookies.getAll()
```

### `delete(name)` {#delete-name}

クッキーの名前を指定して、リクエストからクッキーを削除します。

```ts
// 削除されると true を返し、削除されないと何も返しません
request.cookies.delete('experiments')
```

### `has(name)` {#has-name}

クッキーの名前を指定して、そのクッキーがリクエストに存在する場合は `true` を返します。

```ts
// クッキーが存在する場合は true を返し、存在しない場合は false を返す
request.cookies.has('experiments')
```

### `clear()` {#clear}

リクエストから `Set-Cookie` ヘッダーを削除します。

```ts
request.cookies.clear()
```

## `nextUrl` {#nexturl}

Next.js 特有のプロパティを含む追加の便利なメソッドでネイティブの[`URL`](https://developer.mozilla.org/docs/Web/API/URL)APIを拡張します。

```ts
// /home へのリクエストがある場合、pathname は /home
request.nextUrl.pathname
// /home?name=lee へのリクエストがある場合、searchParams は { 'name': 'lee' }
request.nextUrl.searchParams
```

以下のオプションが利用可能です：

<PagesOnly>

| プロパティ        | 型                        | 説明                                                                                                                                                   |
| ----------------- | ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `basePath`        | `string`                  | URL の[ベースパス](https://nextjs.org/docs/canary/pages/api-reference/config/next-config-js/basePath)。                                                |
| `buildId`         | `string` \| `undefined`   | Next.js アプリケーションのビルド識別子。[カスタマイズ可能](https://nextjs.org/docs/canary/pages/api-reference/config/next-config-js/generateBuildId)。 |
| `defaultLocale`   | `string` \| `undefined`   | [国際化](https://nextjs.org/docs/canary/pages/building-your-application/routing/internationalization)用のデフォルトロケール。                          |
| `domainLocale`    |                           |                                                                                                                                                        |
| - `defaultLocale` | `string`                  | ドメイン内のデフォルトロケール。                                                                                                                       |
| - `domain`        | `string`                  | 特定のロケールに関連付けられたドメイン。                                                                                                               |
| - `http`          | `boolean` \| `undefined`  | ドメインが HTTP を使用しているかどうかを示します。                                                                                                     |
| `locales`         | `string[]` \| `undefined` | 利用可能なロケールの配列。                                                                                                                             |
| `locale`          | `string` \| `undefined`   | 現在アクティブなロケール。                                                                                                                             |
| `url`             | `URL`                     | URL オブジェクト。                                                                                                                                     |

</PagesOnly>

<AppOnly>

| プロパティ     | 型                      | 説明                                                                                                                        |
| -------------- | ----------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| `basePath`     | `string`                | URL の[ベースパス](/docs/app/api-reference/config/next-config-js/basePath)。                                                |
| `buildId`      | `string` \| `undefined` | Next.js アプリケーションのビルド識別子。[カスタマイズ可能](/docs/app/api-reference/config/next-config-js/generateBuildId)。 |
| `pathname`     | `string`                | URL の pathname。                                                                                                           |
| `searchParams` | `Object`                | URL の検索パラメーター。                                                                                                    |

> **注意：** pages router の国際化プロパティは、app router での使用はできません。app router での[国際化について学ぶ](/docs/app/building-your-application/routing/internationalization)。

</AppOnly>
