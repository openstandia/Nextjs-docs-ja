`expireTag` は特定のキャッシュタグに対して、要求に応じて[キャッシュされたデータ](/docs/app/building-your-application/caching)を無効にすることができます。

> **Good to know**:
>
> - `expireTag` は、[Node.js と Edge runtimes](/docs/app/building-your-application/rendering/edge-and-nodejs-runtimes)の両方で利用できます。
> - `expireTag` は、パスが次に訪問されたときにのみキャッシュを無効化します。これは、`expireTag` を動的ルートセグメントで呼び出しても、即座に多くの無効化が発生しないことを意味します。無効化は、パスが次に訪問されたときにのみ行われます。

## リファレンス {#reference}

### パラメータ {#parameters}

```tsx
expireTag(...tags: string[]): void;
```

- `tags`: 再度検証したいデータに関連付けられたキャッシュタグを表す文字列の引数です。各値は256文字以下でなければなりません。値は大文字小文字を区別します。

`fetch`にタグを追加することができます:

```tsx
fetch(url, { next: { tags: [...] } });
```

### 戻り値 {#returns}

`expireTag` は値を返しません。

## 例 {#examples}

### Server Action {#server-action}

Server Action で `expireTag` を呼び出すことができます:

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="app/actions.ts" switcher
'use server'

import { expireTag } from 'next/cache'

export default async function submit() {
  await addPost()
  expireTag('posts', 'blog')
}
```

### ルートハンドラー {#route-handler}

ルートハンドラーで `expireTag` を呼び出すことができます:

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="app/api/revalidate/route.ts" switcher
import type { NextRequest } from 'next/server'
import { expireTag } from 'next/cache'

export async function GET(request: NextRequest) {
  const tag = request.nextUrl.searchParams.get('tag')
  expireTag(tag)
  return Response.json({ revalidated: true, now: Date.now() })
}
```

</TabItem>
