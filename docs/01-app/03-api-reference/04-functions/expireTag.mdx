`expireTag` は、特定のキャッシュタグに対して[キャッシュされたデータ](/docs/app/building-your-application/caching)をオンデマンドでパージすることを可能にします。

> **Good to know**:
>
> - `expireTag` は、[Node.js と Edge ランタイム](/docs/app/building-your-application/rendering/edge-and-nodejs-runtimes)の両方で利用可能です；
> - `expireTag` は、パスが次にアクセスされたときにのみキャッシュを無効化します。よって、dynamic route segment で `expireTag` を呼び出したとしても、すぐに多数の有効期限が切れることはありません。無効化は、パスが次にアクセスされたときにのみ発生します；

## リファレンス {#reference}

### パラメータ {#parameters}

```tsx
expireTag(...tags: string[]): void;
```

- `tags`: 再検証したいデータに関連付けられたキャッシュタグを表す文字列引数です。各タグは256文字以下である必要があります。この値は大文字と小文字を区別します。

`fetch` にタグを追加する方法は次のとおりです:

```tsx
fetch(url, { next: { tags: [...] } });
```

### 戻り値 {#returns}

`expireTag` は値を返しません。

## 例 {#examples}

### Server Action {#server-action}

`expireTag` を Server Action で呼び出すことができます:

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="app/actions.ts" switcher
'use server'

import { expireTag } from 'next/cache'

export default async function submit() {
  await addPost()
  expireTag('posts', 'blog')
}
```

</TabItem>

### ルートハンドラー {#route-handler}

`expireTag` を Route Handler で呼び出すことができます:

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="app/api/revalidate/route.ts" switcher
import type { NextRequest } from 'next/server'
import { expireTag } from 'next/cache'

export async function GET(request: NextRequest) {
  const tag = request.nextUrl.searchParams.get('tag')
  expireTag(tag)
  return Response.json({ revalidated: true, now: Date.now() })
}
```

</TabItem>
