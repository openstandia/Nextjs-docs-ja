---
title: 'typedRoutes'
description: '静的型付きリンクの実験的なサポートを有効化する'
version: '実験的'
---

[静的型付きリンク](/docs/app/api-reference/config/typescript#statically-typed-links) の実験的サポートです。この機能を利用するには、プロジェクトでApp RouterとTypeScriptを使用する必要があります。

```js title="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true, // typedRoutes を有効化する
  },
}

module.exports = nextConfig
```
