`authInterrupts` 設定オプションを使用することで、アプリケーションで [`forbidden`](/docs/app/api-reference/functions/forbidden) および [`unauthorized`](/docs/app/api-reference/functions/unauthorized) API を利用することができます； これらの関数は実験的なものなので、使用するためには `next.config.js` ファイルで `authInterrupts` オプションを有効にする必要があります：

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="next.config.ts" switcher
import type { NextConfig } from 'next'

// nextConfigにauthInterruptsオプションを追加して、実験機能を有効にする
const nextConfig: NextConfig = {
  experimental: {
    authInterrupts: true,
  },
}

export default nextConfig
```

</TabItem>
<TabItem value="js" label="JavaScript">

```js title="next.config.js" switcher
// module.exportsにauthInterruptsオプションを追加して、実験機能を有効にする
module.exports = {
  experimental: {
    authInterrupts: true,
  },
}
```

</TabItem>
