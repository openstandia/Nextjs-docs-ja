`authInterrupts` 設定オプションを使うことで、アプリケーション内で [`forbidden`](/docs/app/api-reference/functions/forbidden) と [`unauthorized`](/docs/app/api-reference/functions/unauthorized) API が使用可能になります。これらの関数は実験的な機能のため、`next.config.js` ファイル内で `authInterrupts` オプションを有効にする必要があります：

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="next.config.ts" switcher
import type { NextConfig } from 'next'

// 実験的な機能を有効にする
const nextConfig: NextConfig = {
  experimental: {
    authInterrupts: true,
  },
}

export default nextConfig
```

</TabItem>
<TabItem value="js" label="JavaScript">

```js title="next.config.js" switcher
// 実験的な機能を有効にする
module.exports = {
  experimental: {
    authInterrupts: true,
  },
}
```

</TabItem>
</Tabs>
