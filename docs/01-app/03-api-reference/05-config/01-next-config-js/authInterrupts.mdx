`authInterrupts` 構成オプションを使用すると、アプリケーションで [`forbidden`](/docs/app/api-reference/functions/forbidden) と [`unauthorized`](/docs/app/api-reference/functions/unauthorized) API を使用できます。これらの関数は実験的であるため、`next.config.js` ファイルで `authInterrupts` オプションを有効にする必要があります：

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="next.config.ts" switcher
import type { NextConfig } from 'next'

// 実験的なオプション 'authInterrupts' を有効にしています
const nextConfig: NextConfig = {
  experimental: {
    authInterrupts: true,
  },
}

export default nextConfig
```

</TabItem>
<TabItem value="js" label="JavaScript">

```js title="next.config.js" switcher
module.exports = {
  experimental: {
    authInterrupts: true,
  },
}
```

</TabItem>
</Tabs>
