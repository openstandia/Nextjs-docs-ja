`authInterrupts`構成オプションを使用することで、アプリケーションで[`forbidden`](/docs/app/api-reference/functions/forbidden)および[`unauthorized`](/docs/app/api-reference/functions/unauthorized) APIを活用できます。これらの関数は実験的であるため、`next.config.js`ファイルで`authInterrupts`オプションを有効にする必要があります：

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="next.config.ts" switcher
import type { NextConfig } from 'next'

// 実験的なauthInterruptsオプションを有効にします
const nextConfig: NextConfig = {
  experimental: {
    authInterrupts: true,
  },
}

export default nextConfig
```

</TabItem>
<TabItem value="js" label="JavaScript">

```js title="next.config.js" switcher
// 実験的なauthInterruptsオプションを有効にします
module.exports = {
  experimental: {
    authInterrupts: true,
  },
}
```

</TabItem>
</Tabs>
