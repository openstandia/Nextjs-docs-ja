`authInterrupts`設定オプションを使用すると、アプリケーションで[`forbidden`](/docs/app/api-reference/functions/forbidden)や[`unauthorized`](/docs/app/api-reference/functions/unauthorized) APIを使用することができます。これらの関数は実験的なものですが、使用するためには`next.config.js`ファイルで`authInterrupts`オプションを有効にする必要があります：

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="next.config.ts" switcher
import type { NextConfig } from 'next'

// experimentalセクションでauthInterruptsをtrueに設定
const nextConfig: NextConfig = {
  experimental: {
    authInterrupts: true,
  },
}

export default nextConfig
```

</TabItem>
<TabItem value="js" label="JavaScript">

```js title="next.config.js" switcher
module.exports = {
  experimental: {
    authInterrupts: true,
  },
} // experimentalセクションでauthInterruptsをtrueに設定
```

</TabItem>
</Tabs>
