---
title: 'typedRoutes'
description: '静的に型付けされたリンクのエクスペリメンタルサポートを有効にします。'
version: 'experimental'
---

[静的に型付けされたリンク](/docs/app/api-reference/config/typescript#statically-typed-links)のエクスペリメンタルサポートです。この機能を利用するには、プロジェクトでApp RouterとTypeScriptを使用する必要があります。

```js title="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
