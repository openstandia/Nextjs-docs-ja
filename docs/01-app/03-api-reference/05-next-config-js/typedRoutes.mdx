---
title: 'typedRoutes'
description: '静的に型付けされたリンクの実験的サポートを有効にする'
version: 'experimental'
---

[静的に型付けされたリンク](/docs/app/api-reference/config/typescript#statically-typed-links)の実験的なサポート。 この機能を使うには、プロジェクトでApp RouterとTypeScriptを使用する必要があります；

```js title="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true, // 静的に型付けされたリンクを有効にする
  },
}

module.exports = nextConfig
```
