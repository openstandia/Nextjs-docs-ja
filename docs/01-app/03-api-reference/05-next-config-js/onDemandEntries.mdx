Next.jsはサーバーが開発中にビルドされたページをどのように解放またはメモリに保持するかを制御するオプションを提供しています。

デフォルト設定を変更するには、`next.config.js`を開き、`onDemandEntries`の設定を追加してください：

```js title="next.config.js"
module.exports = {
  onDemandEntries: {
    // サーバーがページをバッファに保持する期間（ミリ秒単位）
    maxInactiveAge: 25 * 1000,
    // 解放されずに同時に保持されるべきページの数
    pagesBufferLength: 2,
  },
}
```
