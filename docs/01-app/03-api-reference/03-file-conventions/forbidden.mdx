**forbidden** ファイルは、認証中に[`forbidden`](/docs/app/api-reference/functions/forbidden)関数が呼び出されたときにUIをレンダリングするために使用されます。このファイルにより、UIをカスタマイズできると同時に、Next.jsは`403`ステータスコードを返します。

<Tabs>
<TabItem value="tsx" label="TypeScript">

```tsx title="app/forbidden.tsx" switcher
import Link from 'next/link'

export default function Forbidden() {
  return (
    <div>
      <h2>Forbidden</h2>
      <p>このリソースにアクセスする権限がありません。</p>
      <Link href="/">ホームに戻る</Link>
    </div>
  )
}
```

</TabItem>
