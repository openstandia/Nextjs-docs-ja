**forbidden** ファイルは、認証時に [`forbidden`](/docs/app/api-reference/functions/forbidden) 関数が呼び出されるときに UI をレンダリングするために使用されます。UI をカスタマイズできるだけでなく、Next.js は `403` ステータスコードを返します。

<Tabs>
<TabItem value="tsx" label="TypeScript">

```tsx title="app/forbidden.tsx" switcher
import Link from 'next/link'

// アクセス禁止画面のコンポーネントを定義
export default function Forbidden() {
  return (
    <div>
      <h2>Forbidden</h2>
      <p>You are not authorized to access this resource.</p>
      <Link href="/">Return Home</Link> {/* ホームに戻るリンク */}
    </div>
  )
}
```

</TabItem>
