**forbidden**ファイルは、認証中に[`forbidden`](/docs/app/api-reference/functions/forbidden)関数が呼び出された際にUIをレンダリングするために使用されます。UIをカスタマイズすることができるだけでなく、Next.jsは`403`ステータスコードを返します。

<Tabs>
<TabItem value="tsx" label="TypeScript">

```tsx title="app/forbidden.tsx" switcher
import Link from 'next/link'

// Forbiddenコンポーネントのエクスポート
export default function Forbidden() {
  return (
    <div>
      <h2>Forbidden</h2>
      <p>You are not authorized to access this resource.</p>
      <Link href="/">Return Home</Link>
    </div>
  )
}
```

</TabItem>
