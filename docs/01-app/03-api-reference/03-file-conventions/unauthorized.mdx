**unauthorized** ファイルは、認証中に [`unauthorized`](/docs/app/api-reference/functions/unauthorized) 関数が呼び出されたときに UI をレンダリングするために使用されます。UI をカスタマイズできるだけでなく、Next.js は `401` ステータスコードを返します。

<Tabs>
<TabItem value="tsx" label="TypeScript">

```tsx title="app/unauthorized.tsx" switcher
import Login from '@/app/components/Login'

export default function Unauthorized() {
  return (
    <main>
      <h1>401 - Unauthorized</h1>
      <p>このページにアクセスするにはログインしてください。</p>
      <Login />
    </main>
  )
}
```

</TabItem>

## リファレンス {#reference}

### Props {#props}

`unauthorized.js`コンポーネントは、いかなるpropsも受け取りません。

## 例 {#examples}

### 未認証ユーザーにログイン UI を表示する {#displaying-login-ui-to-unauthenticated-users}

[`unauthorized`](/docs/app/api-reference/functions/unauthorized) 関数を用いて、ログイン UI を備えた `unauthorized.js` ファイルをレンダリングすることができます。

<Tabs>
<TabItem value="tsx" label="TypeScript">

```tsx title="app/dashboard/page.tsx" switcher
import { verifySession } from '@/app/lib/dal'
import { unauthorized } from 'next/server'

export default async function DashboardPage() {
  const session = await verifySession()

  if (!session) {
    unauthorized()
  }

  return <div>Dashboard</div>
}
```

</TabItem>
