## バージョン13から14へのアップグレード {#upgrading-from-13-to-14}

Next.jsのバージョン14にアップデートするには、お好みのパッケージマネージャを使って以下のコマンドを実行してください：

```bash title="Terminal"
npm i next@next-14 react@latest react-dom@latest && npm i eslint-config-next@next-14 -D
```

```bash title="Terminal"
yarn add next@next-14 react@latest react-dom@latest && yarn add eslint-config-next@next-14 -D
```

```bash title="Terminal"
pnpm i next@next-14 react@latest react-dom@latest && pnpm i eslint-config-next@next-14 -D
```

```bash title="Terminal"
bun add next@next-14 react@latest react-dom@latest && bun add eslint-config-next@next-14 -D
```

> **Good to know:** TypeScriptを使用している場合は、`@types/react`と`@types/react-dom`も最新のバージョンにアップグレードしてください。

### v14の概要 {#v14-summary}

- 最小のNode.jsバージョンが16.14から18.17に引き上げられました。これは16.xがサポート終了になったためです
- `next export` コマンドは `output: 'export'` 設定に置き換えられました。詳細については [ドキュメント](https://nextjs.org/docs/app/building-your-application/deploying/static-exports) をご覧ください
- `ImageResponse`のための`next/server`インポートは `next/og` にリネームされました。インポートを安全かつ自動でリネームするための[コーデモッドが利用可能です](/docs/app/building-your-application/upgrading/codemods#next-og-import)
- `@next/font`パッケージは組み込みの `next/font` に完全に置き換えられました。インポートを安全かつ自動でリネームするための[コーデモッドが利用可能です](/docs/app/building-your-application/upgrading/codemods#built-in-next-font)
- `next-swc`のWASMターゲットが削除されました
