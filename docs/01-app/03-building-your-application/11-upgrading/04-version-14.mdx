## 13 から 14 へのアップグレード {#upgrading-from-13-to-14}

Next.js のバージョン 14 に更新するには、お好みのパッケージマネージャーを使用して次のコマンドを実行してください：

```bash title="Terminal"
npm i next@next-14 react@latest react-dom@latest && npm i eslint-config-next@next-14 -D
```

```bash title="Terminal"
yarn add next@next-14 react@latest react-dom@latest && yarn add eslint-config-next@next-14 -D
```

```bash title="Terminal"
pnpm i next@next-14 react@latest react-dom@latest && pnpm i eslint-config-next@next-14 -D
```

```bash title="Terminal"
bun add next@next-14 react@latest react-dom@latest && bun add eslint-config-next@next-14 -D
```

> **Good to know:** TypeScript を使用している場合は、`@types/react` と `@types/react-dom` も最新バージョンにアップグレードしてください。

### v14 の概要 {#v14-summary}

- 最小 Node.js バージョンが 16.14 から 18.17 に引き上げられました。16.x は終了したためです
- `next export` コマンドは、`output: 'export'` config に置き換えられました。詳細は[ドキュメント](https://nextjs.org/docs/app/building-your-application/deploying/static-exports)をご覧ください
- `ImageResponse` のための `next/server` のインポートは `next/og` に改名されました。インポートを安全かつ自動的に改名するための[コードモッドが利用可能です](/docs/app/building-your-application/upgrading/codemods#next-og-import)
- `@next/font` パッケージは、組み込みの `next/font` に完全に置き換えられました。インポートを安全かつ自動的に改名するための[コードモッドが利用可能です](/docs/app/building-your-application/upgrading/codemods#built-in-next-font)
- `next-swc` の WASM ターゲットは削除されました
