Next.jsのcanaryチャンネルは、最新の実験的な機能やバグ修正とともに毎日更新されます。安定版のリリース前に新しい機能を試したり、[フィードバックを提供](https://github.com/vercel/next.js/issues)するのに最適な方法です。

Canaryにアップグレードするには、まずNext.jsの最新バージョンを使用していて、すべてが期待通りに動作していることを確認してください。詳しくは、[アップグレードガイド](/docs/app/building-your-application/upgrading)をご覧ください。

その後、次のコマンドを実行します：

```bash title="Terminal"
npm i next@canary
# または {#or}
yarn add next@canary
# または {#or}
pnpm i next@canary
```
