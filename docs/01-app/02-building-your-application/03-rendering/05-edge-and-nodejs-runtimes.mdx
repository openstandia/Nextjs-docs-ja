Next.js では、アプリケーションで使用できる2つのサーバランタイムがあります：

- **Node.js Runtime**（デフォルト）：すべての Node.js API とエコシステムからの互換性のあるパッケージにアクセスできます
- **Edge Runtime**：より限定的な[API のセット](/docs/app/api-reference/edge)を含んでいます

## ユースケース {#use-cases}

- Node.js Runtime は、アプリケーションをレンダリングするために使用されます
- Edge Runtime は、ミドルウェア（リダイレクト、リライト、ヘッダーの設定などのルーティングルール）に使用されます

## 注意事項 {#caveats}

- Edge Runtime はすべての Node.js API をサポートしているわけではありません。一部のパッケージは期待通りに動作しない可能性があります。サポートされていない API については、[Edge Runtime](/docs/app/api-reference/edge#unsupported-apis)で詳しく学びましょう
- Edge Runtime は増分静的再生成（ISR）をサポートしていません
- 両方のランタイムは、展開インフラストラクチャに応じて[ストリーミング](/docs/app/building-your-application/routing/loading-ui-and-streaming)をサポートできます
