## 13 から 14 へのアップグレード {#upgrading-from-13-to-14}

Next.js バージョン 14 へのアップデートは、以下のコマンドを使用してお好みのパッケージマネージャーで実行してください：

```bash title="ターミナル"
npm i next@next-14 react@latest react-dom@latest && npm i eslint-config-next@next-14 -D
```

```bash title="ターミナル"
yarn add next@next-14 react@latest react-dom@latest && yarn add eslint-config-next@next-14 -D
```

```bash title="ターミナル"
pnpm i next@next-14 react@latest react-dom@latest && pnpm i eslint-config-next@next-14 -D
```

```bash title="ターミナル"
bun add next@next-14 react@latest react-dom@latest && bun add eslint-config-next@next-14 -D
```

> **Good to know:** TypeScript を使用している場合は、`@types/react` と `@types/react-dom` も最新バージョンにアップグレードしてください。

### v14 サマリー {#v14-summary}

- Node.js の最小バージョンが 16.14 から 18.17 に引き上げられました。16.x はサポート終了となりました
- `next export` コマンドは、`output: 'export'` 設定に取って代わられました。詳細は[ドキュメント](https://nextjs.org/docs/app/building-your-application/deploying/static-exports)をご覧ください
- `ImageResponse` のための `next/server` インポートは `next/og` にリネームされました。インポートを安全かつ自動的にリネームするための[コードモッドが利用可能です](/docs/app/building-your-application/upgrading/codemods#next-og-import)
- `@next/font` パッケージは、組み込みの `next/font` に完全に移行されました。インポートを安全かつ自動的にリネームするための[コードモッドが利用可能です](/docs/app/building-your-application/upgrading/codemods#built-in-next-font)
- `next-swc` の WASM ターゲットが削除されました
