## 13から14へのアップグレード {#upgrading-from-13-to-14}

Next.jsのバージョン14に更新するには、お好みのパッケージマネージャーを使用して以下のコマンドを実行してください：

```bash title="ターミナル"
npm i next@next-14 react@latest react-dom@latest && npm i eslint-config-next@next-14 -D
```

```bash title="ターミナル"
yarn add next@next-14 react@latest react-dom@latest && yarn add eslint-config-next@next-14 -D
```

```bash title="ターミナル"
pnpm i next@next-14 react@latest react-dom@latest && pnpm i eslint-config-next@next-14 -D
```

```bash title="ターミナル"
bun add next@next-14 react@latest react-dom@latest && bun add eslint-config-next@next-14 -D
```

> **良い情報：** TypeScriptをご使用の場合、`@types/react`と`@types/react-dom`も最新バージョンにアップグレードしてください。

### v14の概要 {#v14-summary}

- 最低Node.jsバージョンが16.14から18.17に引き上げられました；16.xはサポート終了とされました
- `next export`コマンドは`output: 'export'`設定に置き換えられました。詳細は[ドキュメント](https://nextjs.org/docs/app/building-your-application/deploying/static-exports)をご確認ください
- `ImageResponse`用の`next/server`インポートは`next/og`にリネームされました。インポートの安全かつ自動的なリネームには[コードモッドが利用可能](/docs/app/building-your-application/upgrading/codemods#next-og-import)です
- `@next/font`パッケージは組み込みの`next/font`に完全に置き換えられました。インポートの安全かつ自動的なリネームには[コードモッドが利用可能](/docs/app/building-your-application/upgrading/codemods#built-in-next-font)です
- `next-swc`用のWASMターゲットは削除されました
