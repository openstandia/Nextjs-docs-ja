---

Next.jsのcanaryチャンネルは、最新の実験的な機能とバグ修正を含んだ更新が毎日行われています。これは、新機能を安定版リリース前に試し、[フィードバックを提供する](https://github.com/vercel/next.js/issues)のに最適な方法です。

canaryにアップグレードするには、まずあなたのNext.jsが最新バージョンであり、すべてが期待どおりに動作していることを確認してください。詳細は[アップグレードガイド](/docs/app/building-your-application/upgrading)を参照してください。

次に、以下のコマンドを実行してください：

```bash title="Terminal"
npm i next@canary
# または {#or}
yarn add next@canary
# または {#or}
pnpm i next@canary
```

- [`cacheTag`](/docs/app/api-reference/functions/cacheTag)

**認証**:

- [`unauthorized`](/docs/app/api-reference/functions/unauthorized)
