Next.js の canary チャンネルは、最新の実験的な機能やバグ修正で毎日更新されます。安定版でリリースされる前に新機能を試したり、[フィードバックを提供する](https://github.com/vercel/next.js/issues)のに最適です。

canary にアップグレードするには、最新バージョンの Next.js を使用していることを確認し、すべてが期待どおりに動作していることを確認してください。詳細については、[アップグレードガイド](/docs/app/building-your-application/upgrading)を参照してください。

次に、以下のコマンドを実行します：

```bash title="Terminal"
npm i next@canary
# または {#or}
yarn add next@canary
# または {#or}
pnpm i next@canary
```
