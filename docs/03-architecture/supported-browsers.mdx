Next.js は**最新のブラウザ**を設定なしでサポートしています。

- Chrome 64+
- Edge 79+
- Firefox 67+
- Opera 51+
- Safari 12+

## Browserslist {#browserslist}

特定のブラウザや機能をターゲットにしたい場合、Next.js は `package.json` ファイル内で [Browserslist](https://browsersl.ist) の設定をサポートしています。Next.js はデフォルトで次の Browserslist 設定を使用します:

```json title="package.json"
{
  "browserslist": [
    "chrome 64",
    "edge 79",
    "firefox 67",
    "opera 51",
    "safari 12"
  ]
}
```

## Polyfills {#polyfills}

次の[広く使用されている Polyfill](https://github.com/vercel/next.js/blob/canary/packages/next-polyfill-nomodule/src/index.js)を注入します:

- [**fetch()**](https://developer.mozilla.org/docs/Web/API/Fetch_API) — 置換: `whatwg-fetch` と `unfetch`
- [**URL**](https://developer.mozilla.org/docs/Web/API/URL) — 置換: [`url` パッケージ (Node.js API)](https://nodejs.org/api/url.html)
- [**Object.assign()**](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object/assign) — 置換: `object-assign`, `object.assign`, `core-js/object/assign`

もし依存関係がこれらの Polyfill を含んでいる場合、それらは重複を避けるために自動的に本番ビルドから除去されます。

さらに、バンドルサイズを削減するために、Next.js はこれらの Polyfill を必要とするブラウザにのみロードします。世界中の Web トラフィックの大部分はこれらの Polyfill をダウンロードしません。

### Custom Polyfills {#custom-polyfills}

自身のコードや外部の npm 依存関係がターゲットブラウザ（例えば IE 11）でサポートされていない機能を必要とする場合、Polyfill を自分で追加する必要があります。

この場合、必要な**特定の Polyfill**を [Custom `<App>`](https://nextjs.org/docs/canary/pages/building-your-application/routing/custom-app) や個別のコンポーネントでトップレベルのインポートとして追加するべきです。

## JavaScript Language Features {#javascript-language-features}

Next.js は最新の JavaScript 機能をすぐに使用できるようにしています。 [ES6 の機能](https://github.com/lukehoban/es6features)に加えて、Next.js では以下もサポートしています:

- [Async/await](https://github.com/tc39/ecmascript-asyncawait) (ES2017)
- [Object Rest/Spread Properties](https://github.com/tc39/proposal-object-rest-spread) (ES2018)
- [Dynamic `import()`](https://github.com/tc39/proposal-dynamic-import) (ES2020)
- [Optional Chaining](https://github.com/tc39/proposal-optional-chaining) (ES2020)
- [Nullish Coalescing](https://github.com/tc39/proposal-nullish-coalescing) (ES2020)
- [Class Fields](https://github.com/tc39/proposal-class-fields) および [Static Properties](https://github.com/tc39/proposal-static-class-features) (ES2022)
- その他多数！
