次世代のWebアプリケーションフレームワークとして、毎週500万回以上ダウンロードされている Next.js には、世界中に大きくてアクティブな開発者コミュニティがあります。以下の方法でコミュニティに参加できます：

## 貢献

Next.js の開発に貢献する方法は以下のいくつかです：

- [ドキュメント](/docs/community/contribution-guide): 改善の提案や新しいセクションの作成を通じて、ユーザーが Next.js の使い方を理解する手助けをしてください
- [例](https://github.com/vercel/next.js/tree/canary/contributing/examples): Next.js を他のツールやサービスと統合するための新しい例を作成したり、既存のものを改善したりして開発者を支援してください
- [コードベース](https://github.com/vercel/next.js/tree/canary/contributing/core): 基本的なアーキテクチャをより深く学び、バグ修正やエラーの解決、新機能の提案に貢献してください

## 議論

Next.js についての質問がある場合や、他の人を助けたい場合には、いつでも以下の会話に参加できます：

- [GitHub Discussions](https://github.com/vercel/next.js/discussions)
- [Discord](https://discord.com/invite/bUG2bvbtHy)
- [Reddit](https://www.reddit.com/r/nextjs)

## ソーシャルメディア

最新のアップデートを知りたければ [Twitter](https://x.com/nextjs) で Next.js をフォローし、Next.js ビデオを観るために [Vercel YouTube チャンネル](https://www.youtube.com/@VercelHQ) を購読してください。

## 行動規範

私たちは包括的で歓迎するコミュニティの創造を信じています。そのため、すべてのメンバーに [行動規範](https://github.com/vercel/next.js/blob/canary/CODE_OF_CONDUCT.md) を守るようお願いしています。この文書は参加者の行動に関する期待を記載しています。ぜひお読みいただき、安全で尊敬の念に満ちた環境を維持するのにご協力ください。
