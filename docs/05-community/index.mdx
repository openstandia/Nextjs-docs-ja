週に500万回以上ダウンロードされているNext.jsは、世界中の多くの活発な開発者コミュニティを持っています。以下は、私たちのコミュニティに参加する方法です。

## 貢献

Next.jsの開発に貢献する方法はいくつかあります。

- [ドキュメント](/docs/community/contribution-guide)：改善を提案したり、Next.jsの使い方を理解するための新しいセクションを書いたりすることで、ユーザーを助けることができます
- [例](https://github.com/vercel/next.js/tree/canary/contributing/examples)：新しい例を作成したり、既存の例を改善することで、他のツールやサービスと共にNext.jsを統合する方法を開発者に示すことができます
- [コードベース](https://github.com/vercel/next.js/tree/canary/contributing/core)：基盤となるアーキテクチャについて学び、バグ修正やエラーへの貢献、新機能の提案をすることができます

## 議論

Next.jsに関する質問がある場合や他の人を助けたい場合は、いつでも会話に参加することが歓迎されています。

- [GitHub Discussions](https://github.com/vercel/next.js/discussions)
- [Discord](https://discord.com/invite/bUG2bvbtHy)
- [Reddit](https://www.reddit.com/r/nextjs)

## ソーシャルメディア

[Twitter](https://x.com/nextjs)でNext.jsをフォローして最新のアップデートを取得し、[Vercel YouTube チャンネル](https://www.youtube.com/@VercelHQ)を購読してNext.jsの動画をご覧ください。

## 行動規範

私たちは包括的で、歓迎的なコミュニティを作ることを信じています。そのため、全てのメンバーに[行動規範](https://github.com/vercel/next.js/blob/canary/CODE_OF_CONDUCT.md)の遵守をお願いしています。これは、参加者の行動に対する期待を明文化したドキュメントです。どうぞお読みいただき、安全で尊重される環境を維持する手助けをお願いします。
