Next.jsには、アプリケーションで使用できる2つのサーバーランタイムがあります：

- **Node.js Runtime**（デフォルト）で、すべてのNode.js APIとエコシステムからの互換性のあるパッケージにアクセスできます
- **Edge Runtime**で、より限定された[APIセット](/docs/app/api-reference/edge)を含んでいます

## ユースケース

- Node.js runtimeは、アプリケーションをレンダリングするために使用されます
- Edge runtimeは、ミドルウェア（リダイレクト、リライト、ヘッダー設定などのルーティングルール）のために使用されます

## 注意事項

- Edge RuntimeはすべてのNode.js APIをサポートしているわけではありません。一部のパッケージは期待通りに動作しない場合があります。サポートされていないAPIについて詳しくは、[Edge Runtime](/docs/app/api-reference/edge#unsupported-apis)をご覧ください。
- Edge Runtimeは、インクリメンタル静的再生成（ISR）をサポートしていません。
- 両方のランタイムは、デプロイメントインフラストラクチャに応じて、[ストリーミング](/docs/app/building-your-application/routing/loading-ui-and-streaming)をサポートできます。
