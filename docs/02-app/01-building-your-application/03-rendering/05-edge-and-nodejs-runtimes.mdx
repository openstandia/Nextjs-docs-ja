Next.jsは、アプリケーションで使用できる2つのサーバーランタイムを提供しています：

- **Node.js Runtime**（デフォルト）は、すべてのNode.js APIとエコシステムからの互換性のあるパッケージにアクセスできます
- より限られた[APIのセット](/docs/app/api-reference/edge)を含む**Edge Runtime**

## ユースケース {#use-cases}

- Node.js Runtimeはアプリケーションのレンダリングに使用されます
- Edge Runtimeはミドルウェア（リダイレクト、リライト、およびヘッダー設定のようなルーティングルール）に使用されます

## 注意事項 {#caveats}

- Edge RuntimeはすべてのNode.js APIをサポートしているわけではありません。いくつかのパッケージは期待通りに動作しない場合があります。未対応のAPIについて詳しくは、[Edge Runtime](/docs/app/api-reference/edge#unsupported-apis)で確認してください
- Edge RuntimeはIncremental Static Regeneration (ISR)をサポートしていません
- 両方のランタイムは、デプロイインフラストラクチャに応じて[ストリーミング](/docs/app/building-your-application/routing/loading-ui-and-streaming)をサポートできます
