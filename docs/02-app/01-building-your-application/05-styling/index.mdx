Next.js は、アプリケーションをスタイリングするためのさまざまな方法をサポートしています：

- **CSS Modules**: 名前の競合を避け、メンテナンス性を向上させるために、ローカルにスコープされた CSS クラスを作成します
- **Global CSS**: 伝統的な CSS に慣れている人にとって使いやすく、シンプルですが、アプリケーションが成長するにつれて CSS バンドルが大きくなり、スタイルの管理が難しくなる可能性があります
- **Tailwind CSS**: ユーティリティファーストの CSS フレームワークであり、ユーティリティクラスを組み合わせることで迅速なカスタムデザインを可能にします
- **Sass**: 変数、ネストされたルール、ミキシンなどの機能を提供する人気のある CSS プリプロセッサです
- **CSS-in-JS**: JavaScript コンポーネント内に直接 CSS を埋め込み、動的かつスコープされたスタイリングを可能にします
