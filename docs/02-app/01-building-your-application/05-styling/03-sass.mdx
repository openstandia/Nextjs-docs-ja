Next.jsは、`.scss`および`.sass`拡張子のパッケージをインストールした後でSassを統合するための組み込みサポートを提供しています。CSS Modulesを使用したコンポーネントレベルのSassおよび`.module.scss`または`.module.sass`拡張子を使用することができます。

まず、[`sass`](https://github.com/sass/sass)をインストールします：

```bash filename="Terminal"
npm install --save-dev sass
```

> **Good to know**:
>
> Sassは、それぞれ独自の拡張子を持つ[2つの異なる構文](https://sass-lang.com/documentation/syntax)をサポートしています。
> `.scss`拡張子は[SCSS構文](https://sass-lang.com/documentation/syntax#scss)を使用する必要があります、
> 一方、`.sass`拡張子は[インデント構文 ("Sass")](https://sass-lang.com/documentation/syntax#the-indented-syntax)を使用する必要があります。
>
> どちらを選択するか迷った場合は、CSSのスーパーセットであるため、
> インデント構文 ("Sass")を学習する必要がない`.scss`拡張子から始めることをお勧めします。

### Sassオプションのカスタマイズ

Sassオプションを設定したい場合は、`next.config`内で`sassOptions`を使用します。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  sassOptions: {
    additionalData: `$var: red;`,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
/** @type {import('next').NextConfig} */

const nextConfig = {
  sassOptions: {
    additionalData: `$var: red;`,
  },
}

module.exports = nextConfig
```

#### 実装

使用するSassの実装を指定するには、`implementation`プロパティを使用できます。デフォルトでは、Next.jsは[`sass`](https://www.npmjs.com/package/sass)パッケージを使用します。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  sassOptions: {
    implementation: 'sass-embedded',
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
/** @type {import('next').NextConfig} */

const nextConfig = {
  sassOptions: {
    implementation: 'sass-embedded',
  },
}

module.exports = nextConfig
```

### Sass変数

Next.jsは、CSS ModuleファイルからエクスポートされたSass変数をサポートしています。

例えば、エクスポートされた`primaryColor`Sass変数を使用する場合：

```scss filename="app/variables.module.scss"
$primary-color: #64ff00;

:export {
  primaryColor: $primary-color;
}
```

<AppOnly>

```jsx filename="app/page.js"
// ルート `/` URLにマップされる

import variables from './variables.module.scss'

export default function Page() {
  return <h1 style={{ color: variables.primaryColor }}>Hello, Next.js!</h1>
}
```

</AppOnly>

<PagesOnly>

```jsx filename="pages/_app.js"
import variables from '../styles/variables.module.scss'

export default function MyApp({ Component, pageProps }) {
  return (
    <Layout color={variables.primaryColor}>
      <Component {...pageProps} />
    </Layout>
  )
}
```

</PagesOnly>
