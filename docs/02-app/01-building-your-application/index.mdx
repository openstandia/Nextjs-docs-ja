Next.js は柔軟でフルスタックな Web アプリケーションを作成するための構成要素を提供します。**アプリケーションの構築**に関するガイドでは、これらの機能をどのように使用し、アプリケーションの動作をどのようにカスタマイズするかを説明します。

セクションとページは基本から応用まで順序立てて整理されているため、Next.js アプリケーションの構築時にステップバイステップで進むことができます。ただし、お好みの順序で読んだり、適用可能なケースのページに直接進むこともできます。

<AppOnly>

Next.js が初めての方には、[ルーティング](/docs/app/building-your-application/routing)、[レンダリング](/docs/app/building-your-application/rendering)、[データの取得](/docs/app/building-your-application/data-fetching)、および[スタイリング](/docs/app/building-your-application/styling)のセクションを最初に読むことをお勧めします。これらのセクションは、Next.js の基本的な概念と Web 概念を紹介し、開始の手助けをします。その後、[最適化](/docs/app/building-your-application/optimizing)や[設定](/docs/app/building-your-application/configuring)といった他のセクションをさらに深く掘り下げることができます。そして、準備が整ったら、[デプロイ](/docs/app/building-your-application/deploying)および[アップグレード](/docs/app/building-your-application/upgrading)セクションをチェックしてください。

</AppOnly>
