Next.js は、柔軟でフルスタックなWebアプリケーションを作成するための構成要素を提供します。**アプリケーションの構築**に関するガイドは、これらの機能をどのように使用し、アプリケーションの挙動をどのようにカスタマイズするかを説明しています。

セクションとページは基本から高度なものまで順に整理されており、Next.js アプリケーションを構築する際にステップ・バイ・ステップで従うことができます。しかし、この順番に沿って読まなくても、それぞれのユースケースに適したページにスキップすることも可能です。

<AppOnly>

Next.js に初めて触れる方は、[ルーティング](/docs/app/building-your-application/routing)、[レンダリング](/docs/app/building-your-application/rendering)、[データ取得](/docs/app/building-your-application/data-fetching)、[スタイリング](/docs/app/building-your-application/styling)のセクションから始めることをお勧めします。これらはNext.js とWebの基本概念を紹介しており、スタートを切るのに役立ちます。その後、[最適化](/docs/app/building-your-application/optimizing) や [設定](/docs/app/building-your-application/configuring) といった他のセクションに深入りすることができます。最後に、準備ができたら [デプロイ](/docs/app/building-your-application/deploying) と [アップグレード](/docs/app/building-your-application/upgrading) のセクションを確認してください。

</AppOnly>
