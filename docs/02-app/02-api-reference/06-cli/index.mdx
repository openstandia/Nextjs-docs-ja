Next.js には **2 つの** コマンドラインインターフェース (CLI) ツールが付属しています；

- **`create-next-app`**: デフォルトのテンプレートや、公開されている GitHub リポジトリの[例](https://github.com/vercel/next.js/tree/canary/examples)を使用して、新しい Next.js アプリケーションを迅速に作成します
- **`next`**: Next.js の開発サーバーを実行し、アプリケーションをビルドするなど、さまざまな機能があります
