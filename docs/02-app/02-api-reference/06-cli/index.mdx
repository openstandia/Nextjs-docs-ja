Next.js には**2つの**コマンドラインインターフェース (CLI) ツールがあります；

- **`create-next-app`**: デフォルトのテンプレートや公開されているGithubリポジトリの[例](https://github.com/vercel/next.js/tree/canary/examples)を使用して、新しいNext.jsアプリケーションをすばやく作成します
- **`next`**: Next.js 開発サーバを実行し、アプリケーションをビルドするなどの機能を提供します
