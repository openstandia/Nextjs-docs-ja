Next.js には**2つ**のコマンドラインインターフェース（CLI）ツールがあります：

- **`create-next-app`**: デフォルトのテンプレートや公開GitHubリポジトリからの[例](https://github.com/vercel/next.js/tree/canary/examples)を使用して、新しいNext.jsアプリケーションを素早く作成します
- **`next`**: Next.js 開発サーバーを実行したり、アプリケーションをビルドしたりするためのツールです
