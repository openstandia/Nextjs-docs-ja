Route Handlersは、Web [Request](https://developer.mozilla.org/docs/Web/API/Request)と[Response](https://developer.mozilla.org/docs/Web/API/Response) APIを使用して、特定のルート用にカスタムのリクエストハンドラを作成することを可能にします。

## リファレンス

### HTTPメソッド

**route** ファイルでは、特定のルート用にカスタムリクエストハンドラを作成できます。以下の[HTTPメソッド](https://developer.mozilla.org/docs/Web/HTTP/Methods)がサポートされています：`GET`、`POST`、`PUT`、`PATCH`、`DELETE`、`HEAD`、`OPTIONS`。

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="route.ts" switcher
export async function GET(request: Request) {}

export async function HEAD(request: Request) {}

export async function POST(request: Request) {}

export async function PUT(request: Request) {}

export async function DELETE(request: Request) {}

export async function PATCH(request: Request) {}

// `OPTIONS` が定義されていない場合、Next.js は自動的に `OPTIONS` を実装し、Route Handlerで他に定義されているメソッドに応じて適切なResponse `Allow`ヘッダーを設定します。
export async function OPTIONS(request: Request) {}
```
</TabItem>
<TabItem value="js" label="JavaScript">

```js title="route.js" switcher
export async function GET(request) {}

export async function HEAD(request) {}

export async function POST(request) {}

export async function PUT(request) {}

export async function DELETE(request) {}

export async function PATCH(request) {}

// `OPTIONS` が定義されていない場合、Next.js は自動的に `OPTIONS` を実装し、Route Handlerで他に定義されているメソッドに応じて適切なResponse `Allow`ヘッダーを設定します。
export async function OPTIONS(request) {}
```
</TabItem>
</Tabs>


### パラメータ

#### `request` (任意)

`request` オブジェクトは、Web [Request](https://developer.mozilla.org/docs/Web/API/Request) APIの拡張である[NextRequest](/docs/app/api-reference/functions/next-request)オブジェクトです。`NextRequest` では、`cookies`に簡単にアクセスできることや、拡張され解析されたURLオブジェクト `nextUrl` など、受信リクエストのさらなる制御が可能です。

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="route.ts" switcher
import type { NextRequest } from 'next/server'

export async function GET(request: NextRequest) {
  const url = request.nextUrl
}
```
</TabItem>
<TabItem value="js" label="JavaScript">

```js title="route.js" switcher
export async function GET(request) {
  const url = request.nextUrl
}
```
</TabItem>
</Tabs>


#### `context` (任意)

- **`params`**: 現在のルートに対する[動的ルートのパラメータ](/docs/app/building-your-application/routing/dynamic-routes)を含むオブジェクトに解決されるpromiseです

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="app/dashboard/[team]/route.ts" switcher
export async function GET(
  request: Request,
  { params }: { params: Promise<{ team: string }> }
) {
  const team = (await params).team
}
```
</TabItem>
<TabItem value="js" label="JavaScript">

```js title="app/dashboard/[team]/route.js" switcher
export async function GET(request, { params }) {
  const team = (await params).team
}
```
</TabItem>
</Tabs>


| 例                                 | URL            | `params`                           |
| -------------------------------- | -------------- | ---------------------------------- |
| `app/dashboard/[team]/route.js`  | `/dashboard/1` | `Promise<{ team: '1' }>`           |
| `app/shop/[tag]/[item]/route.js` | `/shop/1/2`    | `Promise<{ tag: '1', item: '2' }>` |
| `app/blog/[...slug]/route.js`    | `/blog/1/2`    | `Promise<{ slug: ['1', '2'] }>`    |

### クッキーの処理

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="route.ts" switcher
import { cookies } from 'next/headers'

export async function GET(request: NextRequest) {
  const cookieStore = await cookies()

  const a = cookieStore.get('a')
  const b = cookieStore.set('b', '1')
  const c = cookieStore.delete('c')
}
```
</TabItem>
<TabItem value="js" label="JavaScript">

```js title="route.js" switcher
import { cookies } from 'next/headers'

export async function GET(request) {
  const cookieStore = await cookies()

  const a = cookieStore.get('a')
  const b = cookieStore.set('b', '1')
  const c = cookieStore.delete('c')
}
```
