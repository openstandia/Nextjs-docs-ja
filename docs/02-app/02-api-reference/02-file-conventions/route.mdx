Route Handlersを使用すると、Web [Request](https://developer.mozilla.org/docs/Web/API/Request)と[Response](https://developer.mozilla.org/docs/Web/API/Response) APIを使って特定のルートに対するカスタムリクエストハンドラを作成することができます。

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="route.ts" switcher
export async function GET() {
  return Response.json({ message: 'Hello World' })
}
```

</TabItem>
<TabItem value="js" label="JavaScript">

```js title="route.js" switcher
export async function GET() {
  return Response.json({ message: 'Hello World' })
}
```

</TabItem>
</Tabs>

## リファレンス {#reference}

### HTTPメソッド {#http-methods}

**route** ファイルを使用すると、特定のルートに対するカスタムリクエストハンドラを作成することができます。サポートされている[HTTPメソッド](https://developer.mozilla.org/docs/Web/HTTP/Methods)は、`GET`、`POST`、`PUT`、`PATCH`、`DELETE`、`HEAD`、および `OPTIONS` です。

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="route.ts" switcher
export async function GET(request: Request) {}

export async function HEAD(request: Request) {}

export async function POST(request: Request) {}

export async function PUT(request: Request) {}

export async function DELETE(request: Request) {}

export async function PATCH(request: Request) {}

// `OPTIONS`が定義されていない場合、Next.jsは他のメソッドに応じて適切なResponse `Allow`ヘッダーを設定し、自動的に`OPTIONS`を実装します。
export async function OPTIONS(request: Request) {}
```

</TabItem>
<TabItem value="js" label="JavaScript">

```js title="route.js" switcher
export async function GET(request) {}

export async function HEAD(request) {}

export async function POST(request) {}

export async function PUT(request) {}

export async function DELETE(request) {}

export async function PATCH(request) {}

// `OPTIONS`が定義されていない場合、Next.jsは他のメソッドに応じて適切なResponse `Allow`ヘッダーを設定し、自動的に`OPTIONS`を実装します。
export async function OPTIONS(request) {}
```

</TabItem>
</Tabs>

### パラメータ {#parameters}

#### `request` (オプション) {#request-optional}

`request` オブジェクトは、Web [Request](https://developer.mozilla.org/docs/Web/API/Request) APIの拡張である [NextRequest](/docs/app/api-reference/functions/next-request) オブジェクトです。`NextRequest` は、`cookies` に簡単にアクセスできることや、拡張された解析済みURLオブジェクト `nextUrl` など、受信リクエストをより詳細に制御できるようにします。

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="route.ts" switcher
import type { NextRequest } from 'next/server'

export async function GET(request: NextRequest) {
  const url = request.nextUrl
}
```

</TabItem>
<TabItem value="js" label="JavaScript">

```js title="route.js" switcher
export async function GET(request) {
  const url = request.nextUrl
}
```

</TabItem>
</Tabs>

#### `context` (オプション) {#context-optional}

- **`params`**: 現在のルートの[動的ルートパラメータ](/docs/app/building-your-application/routing/dynamic-routes)を含むオブジェクトに解決されるプロミスです。

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="app/dashboard/[team]/route.ts" switcher
export async function GET(
  request: Request,
  { params }: { params: Promise<{ team: string }> }
) {
  const team = (await params).team
}
```

</TabItem>
<TabItem value="js" label="JavaScript">

```js title="app/dashboard/[team]/route.js" switcher
export async function GET(request, { params }) {
  const team = (await params).team
}
```

</TabItem>
</Tabs>

| 例                               | URL            | `params`                           |
| -------------------------------- | -------------- | ---------------------------------- |
| `app/dashboard/[team]/route.js`  | `/dashboard/1` | `Promise<{ team: '1' }>`           |
| `app/shop/[tag]/[item]/route.js` | `/shop/1/2`    | `Promise<{ tag: '1', item: '2' }>` |
| `app/blog/[...slug]/route.js`    | `/blog/1/2`    | `Promise<{ slug: ['1', '2'] }>`    |

## 例 {#examples}

### cookieの処理方法 {#handling-cookies}

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="route.ts" switcher
import { cookies } from 'next/headers'

export async function GET(request: NextRequest) {
  const cookieStore = await cookies()

  const a = cookieStore.get('a')
  const b = cookieStore.set('b', '1')
  const c = cookieStore.delete('c')
}
```

</TabItem>
<TabItem value="js" label="JavaScript">

```js title="route.js" switcher
import { cookies } from 'next/headers'

export async function GET(request) {
  const cookieStore = await cookies()

  const a = cookieStore.get('a')
  const b = cookieStore.set('b', '1')
  const c = cookieStore.delete('c')
}
```
