`use client` 指令は、コンポーネントを**クライアントサイド**でレンダリングすることを指定します。これは、クライアントサイドのJavaScript機能（状態管理、イベント処理、ブラウザーAPIのアクセスなど）を必要とするインタラクティブなユーザーインターフェース（UI）を作成する場合に使用します。これはReactの機能です。

## 使用法 {#usage}

コンポーネントをClient Componentとして指定するには、`use client` 指令を **ファイルの最上部** に、すべてのインポートより前に追加します：

<Tabs>
<TabItem value="tsx" label="TypeScript">

```tsx title="app/components/counter.tsx" highlight={1} switcher
'use client'

import { useState } from 'react'

export default function Counter() {
  const [count, setCount] = useState(0)

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  )
}
```

</TabItem>
<TabItem value="jsx" label="JavaScript">

```jsx title="app/components/counter.js" highlight={1} switcher

import { useState } from 'react'

export default function Counter() {
  const [count, setCount] = useState(0)

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  )
}
```

</TabItem>

## Server Components 内に Client Components をネストする {#nesting-client-components-within-server-components}

Server Components と Client Components を組み合わせることで、パフォーマンスとインタラクティブ性を兼ね備えたアプリケーションを構築できます：

1. **Server Components**：静的コンテンツ、データ取得、SEOに優れた要素に使用します
2. **Client Components**：状態、エフェクト、ブラウザーAPIを必要とするインタラクティブな要素に使用します
3. **コンポーネントの構成**：サーバーロジックとクライアントロジックを明確に分離するために、必要に応じてServer Components内にClient Componentsをネストします

次の例では：

- `Header` は静的コンテンツを処理するServer Componentです
- `Counter` はページ内でのインタラクティビティを有効にするClient Componentです

<Tabs>
<TabItem value="tsx" label="TypeScript">

```tsx title="app/page.tsx" highlight={2,8} switcher
import Header from './header'
import Counter from './counter' // これはClient Componentです

export default function Page() {
  return (
    <div>
      <Header />
      <Counter />
    </div>
  )
}
```

</TabItem>
<TabItem value="jsx" label="JavaScript">

```jsx title="app/page.js" highlight={2,8} switcher
import Header from './header'
import Counter from './counter' // これはClient Componentです

export default function Page() {
  return (
    <div>
      <Header />
      <Counter />
    </div>
  )
}
```

</TabItem>
</Tabs>
