Next.jsはローカルパッケージ（モノレポのような）や外部依存関係（`node_modules`）から依存関係を自動的にトランスパイルおよびバンドルできます。これは`next-transpile-modules`パッケージを置き換えます。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
// パッケージ名をトランスパイルする
const nextConfig = {
  transpilePackages: ['package-name'],
}

module.exports = nextConfig
```
