Next.js は、ローカルパッケージ（monorepo など）または外部の依存関係（`node_modules`）から依存関係を自動的にトランスパイルおよびバンドルすることができます。これにより、`next-transpile-modules`パッケージが置き換えられます。

```js title="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['package-name'],
}

module.exports = nextConfig
```
