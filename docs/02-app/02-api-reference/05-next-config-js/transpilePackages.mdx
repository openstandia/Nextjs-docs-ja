Next.jsはローカルパッケージ（モノレポのような）または外部依存関係（`node_modules`）から依存関係を自動でトランスパイルおよびバンドルできます。これは `next-transpile-modules` パッケージを置き換えます。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['package-name'], // パッケージ名
}

module.exports = nextConfig
```
