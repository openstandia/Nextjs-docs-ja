実験的な`instrumentationHook`オプションにより、Next.jsアプリで[`instrumentation`ファイル](/docs/app/api-reference/file-conventions/instrumentation)を介してインスツルメンテーションを設定することができます。

```js title="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true, // インスツルメンテーションフックを有効にする
  },
}
```

| バージョン | 変更内容                                                    |
| --------- | ----------------------------------------------------------- |
| `v13.2.0` | `instrumentationHook`が実験的な機能として導入されました       |
