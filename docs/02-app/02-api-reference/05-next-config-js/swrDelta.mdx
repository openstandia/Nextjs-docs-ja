CDNがISR対応ページを処理するための`Cache-Control`ヘッダーにおいて、カスタムのstale-while-revalidate期間を指定することができます。

`next.config.js`を開き、`swrDelta`を設定に追加します：

```js title="next.config.js"
module.exports = {
  // 1年間を秒で指定
  swrDelta: 31536000,
}
```

これで、`Cache-Control`ヘッダーにデフォルトの1年ではなく、カスタムのstale-while-revalidate期間が含まれるようになります。
