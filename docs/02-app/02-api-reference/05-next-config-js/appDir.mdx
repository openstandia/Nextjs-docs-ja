---
title: 'appDir'
description: 'レイアウトやストリーミングなどを使用するためにApp Routerを有効にします'
---

> **知っておくとよいこと**: Next.js 13.4から、このオプションは**不要**になりました；App Routerは安定しています。

App Router（[`app` ディレクトリ](/docs/app/building-your-application/routing)）は、[レイアウト](/docs/app/building-your-application/routing/layouts-and-templates)、[server component](/docs/app/building-your-application/rendering/server-components)、[ストリーミング](/docs/app/building-your-application/routing/loading-ui-and-streaming)、および[データフェッチのコロケーション](/docs/app/building-your-application/data-fetching)のサポートを可能にします。

`app` ディレクトリを使用すると、自動的に[React Strict Mode](https://react.dev/reference/react/StrictMode)が有効になります；[`app`を段階的に採用する方法を学ぶ](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app)ことができます。
