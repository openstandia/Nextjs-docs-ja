---
title: 'appDir'
description: 'App Router を有効にして root レイアウト、ストリーミングなどを使用します'
---

> **Good to know**: このオプションは Next.js 13.4 以降では**不要**になりました; App Router は現在安定しています。

App Router（[`app` ディレクトリ](/docs/app/building-your-application/routing)）は、[root レイアウト](/docs/app/building-your-application/routing/layouts-and-templates)、[server component](/docs/app/building-your-application/rendering/server-components)、[ストリーミング](/docs/app/building-your-application/routing/loading-ui-and-streaming)、および[共同配置されたデータフェッチ](/docs/app/building-your-application/data-fetching)のサポートを可能にします。

`app` ディレクトリを使用すると、自動的に [React Strict Mode](https://react.dev/reference/react/StrictMode) が有効になります; [`app` を徐々に採用する方法を学びます](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app)。
