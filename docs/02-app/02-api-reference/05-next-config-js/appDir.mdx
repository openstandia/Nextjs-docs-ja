---
title: appDir
description: レイアウト、ストリーミングなどを利用するためのAppルーターを有効にする。
---

> **Good to know**: このオプションはNext.js 13.4以降、**不要**になりました。Appルーターは現在安定しています。

Appルーター（[`app`ディレクトリ](/docs/app/building-your-application/routing)）は、[レイアウト](/docs/app/building-your-application/routing/layouts-and-templates)、[サーバーコンポーネント](/docs/app/building-your-application/rendering/server-components)、[ストリーミング](/docs/app/building-your-application/routing/loading-ui-and-streaming)、および[コロケートされたデータ取得](/docs/app/building-your-application/data-fetching)のサポートを有効にします。

`app`ディレクトリを使用すると、[Reactストリクトモード](https://react.dev/reference/react/StrictMode) が自動的に有効になります。[`app`を段階的に採用する方法](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app)を学びましょう。
