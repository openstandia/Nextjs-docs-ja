---
title: mdxRs
description: App RouterでMDXファイルをコンパイルするための新しいRustコンパイラを使用します。
version: experimental。
---

`@next/mdx`での実験的な使用のために。新しいRustコンパイラを使用してMDXファイルをコンパイルします。

```js filename="next.config.js"
const withMDX = require('@next/mdx')()

/** @type {import('next').NextConfig} */
// ページ拡張子としてts、tsx、mdxを指定
const nextConfig = {
  pageExtensions: ['ts', 'tsx', 'mdx'],
  experimental: {
    mdxRs: true, // mdxRsを有効化
  },
}

// MDXサポートを次の設定に適用
module.exports = withMDX(nextConfig)
```
