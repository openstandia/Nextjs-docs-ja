---
title: 'mdxRs'
description: '新しいRustコンパイラを使用してApp RouterでMDXファイルをコンパイルします。'
version: 'experimental。'
---

`@next/mdx`を使用する実験的な用途に適しています。新しいRustコンパイラを使用してMDXファイルをコンパイルします。

```js title="next.config.js"
const withMDX = require('@next/mdx')()

/** @type {import('next').NextConfig} */
const nextConfig = {
  pageExtensions: ['ts', 'tsx', 'mdx'],
  experimental: {
    mdxRs: true,
  },
}

module.exports = withMDX(nextConfig)
```
