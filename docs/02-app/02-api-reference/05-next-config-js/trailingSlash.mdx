デフォルトでは、Next.jsはトレイリングスラッシュを含むURLを、それがないURLにリダイレクトします。例えば、`/about/`は`/about`にリダイレクトされます。この挙動を逆の方法で設定することも可能で、トレイリングスラッシュを含まないURLを、それがあるURLにリダイレクトするようにできます。

`next.config.js`を開き、`trailingSlash`設定を追加してください：

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
}
```

このオプションを設定すると、`/about`のようなURLは`/about/`にリダイレクトされるようになります。

[`output: "export"`](/docs/app/building-your-application/deploying/static-exports)設定と一緒に使用すると、`/about`ページは`/about/index.html`を出力します（デフォルトでは`/about.html`）。
