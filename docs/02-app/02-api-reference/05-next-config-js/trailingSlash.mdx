デフォルトでは、Next.jsはトレーリングスラッシュがあるURLを、トレーリングスラッシュがない対応するものにリダイレクトします。例えば、`/about/`は`/about`にリダイレクトされます。この動作を逆に設定し、トレーリングスラッシュがないURLを、トレーリングスラッシュがある対応するURLにリダイレクトするように構成できます。

`next.config.js`を開いて、`trailingSlash`設定を追加します：

```js title="next.config.js"
module.exports = {
  trailingSlash: true,
}
```

このオプションを設定すると、`/about`のようなURLが`/about/`にリダイレクトされます。

[`output: "export"`](/docs/app/building-your-application/deploying/static-exports)構成と一緒に使用すると、`/about`ページはデフォルトの`/about.html`ではなく、`/about/index.html`を出力します。
