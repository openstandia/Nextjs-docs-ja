`crossOrigin`オプションを使用して、すべての`<script>`タグに[`crossOrigin`属性](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) を追加します<AppOnly>[`next/script`](/docs/app/building-your-application/optimizing/scripts)コンポーネント</AppOnly> <PagesOnly>[`next/script`](/docs/pages/building-your-application/optimizing/scripts)および[`next/head`](/docs/pages/api-reference/components/head)コンポーネント</PagesOnly>において、クロスオリジンリクエストの処理方法を定義します。

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## オプション

- `'anonymous'`: [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous)属性を追加します
- `'use-credentials'`: [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials)を追加します
