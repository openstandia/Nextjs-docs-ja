{/* このドキュメントの内容は、app router と pages router の間で共有されています。Pages Router に特有の内容を追加するには、`<PagesOnly>Content</PagesOnly>` コンポーネントを使用できます。共有されるコンテンツはコンポーネントで囲まないようにしてください。 */}

`crossOrigin` オプションを使用して、<AppOnly>[`next/script`](/docs/app/building-your-application/optimizing/scripts) コンポーネント</AppOnly> <PagesOnly>[`next/script`](https://nextjs.org/docs/canary/pages/building-your-application/optimizing/scripts) コンポーネントおよび [`next/head`](https://nextjs.org/docs/canary/pages/api-reference/components/head)コンポーネント</PagesOnly>によって生成されるすべての `<script>` タグに [`crossOrigin` 属性](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) を追加し、クロスオリジンリクエストをどのように処理するかを定義します。

```js title="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## オプション {#options}

- `'anonymous'`： [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous) 属性を追加します
- `'use-credentials'`： [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials) 属性を追加します
