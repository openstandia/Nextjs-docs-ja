`crossOrigin` オプションを使って、すべての `<script>` タグに [`crossOrigin` 属性](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) を追加し、<AppOnly>[`next/script`](/docs/app/building-your-application/optimizing/scripts) コンポーネントに対応</AppOnly> <PagesOnly>[`next/script`](/docs/pages/building-your-application/optimizing/scripts)と[`next/head`](/docs/pages/api-reference/components/head)コンポーネントに対応</PagesOnly>し、クロスオリジンリクエストがどのように処理されるべきかを定義します。

```js title="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## オプション

- `'anonymous'`: [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous) 属性を追加
- `'use-credentials'`: [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials) 属性を追加
