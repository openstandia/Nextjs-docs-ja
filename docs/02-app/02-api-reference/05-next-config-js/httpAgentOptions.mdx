Node.jsのバージョン18より前では、Next.jsは自動的に`fetch()`を[undici](/docs/architecture/supported-browsers#polyfills)でポリフィルし、デフォルトで[HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive)を有効にします。

サーバーサイドでのすべての`fetch()`呼び出しに対してHTTP Keep-Aliveを無効にするには、`next.config.js`を開き、`httpAgentOptions`設定を追加してください：

```js title="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false, // Keep-Aliveを無効にします
  },
}
```
