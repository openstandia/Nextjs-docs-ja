Node.jsのバージョンが18より前の場合、Next.jsは自動的に`fetch()`を[undici](/docs/architecture/supported-browsers#polyfills)でポリフィルし、デフォルトで[HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive)を有効にします。

サーバーサイドのすべての`fetch()`呼び出しに対してHTTP Keep-Aliveを無効にするには、`next.config.js`を開いて、`httpAgentOptions`の設定を追加します：

```js title="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
