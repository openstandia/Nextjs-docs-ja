> **Good to know**: Next.js 13.5.1以降、Strict Modeは`app` routerではデフォルトで`true`になっているため、上記の設定は`pages`のみ必要です。Strict Modeを無効にするには、`reactStrictMode: false`を設定すれば可能です。

> **Suggested**: Next.jsアプリケーションでStrict Modeを有効にすることを強くお勧めします。これにより、Reactの将来に備えてアプリケーションがより準備されます。

Reactの[Strict Mode](https://react.dev/reference/react/StrictMode)は、アプリケーションの潜在的な問題を強調するための開発モード専用の機能です。不安全なライフサイクルやレガシーAPIの使用、その他多くの機能を特定するのに役立ちます。

Next.jsのランタイムはStrict Modeに準拠しています。Strict Modeにオプトインするには、`next.config.js`で次の設定を行ってください：

```js filename="next.config.js"
module.exports = {
  reactStrictMode: true,
}
```

あなたまたはあなたのチームがアプリケーション全体でStrict Modeを使用する準備ができていなくても大丈夫です！`<React.StrictMode>`を使用して、ページごとに段階的に移行することができます。
