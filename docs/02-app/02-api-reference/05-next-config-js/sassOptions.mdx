---
title: sassOptions
description: Sassオプションを設定する。
---

`sassOptions`を使ってSassコンパイラを設定します。

```ts title="next.config.ts" switcher
import type { NextConfig } from 'next'

const sassOptions = {
  additionalData: `
    $var: red;
  `, // 追加のSassデータをここに入力
}

const nextConfig: NextConfig = {
  sassOptions: {
    ...sassOptions,
    implementation: 'sass-embedded', // Sass実装を指定する
  },
}

export default nextConfig
```

```js title="next.config.js" switcher
/** @type {import('next').NextConfig} */

const sassOptions = {
  additionalData: `
    $var: red;
  `, // 追加のSassデータをここに入力
}

const nextConfig = {
  sassOptions: {
    ...sassOptions,
    implementation: 'sass-embedded', // Sass実装を指定する
  },
}

module.exports = nextConfig
```

> **Good to know:** `sassOptions`は`implementation`以外のプロパティは型定義されていません。これはNext.jsが他の可能なプロパティを保持していないためです。
