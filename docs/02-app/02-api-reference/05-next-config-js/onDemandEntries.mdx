Next.jsは、開発環境でビルドされたページをサーバーがどのように破棄またはメモリに保持するかを制御するためのオプションを提供します。

デフォルトを変更するには、`next.config.js`を開いて`onDemandEntries`の設定を追加してください：

```js title="next.config.js"
module.exports = {
  onDemandEntries: {
    // サーバーがページをバッファに保持する期間（ミリ秒）
    maxInactiveAge: 25 * 1000,
    // 破棄されることなく同時に保持されるべきページの数
    pagesBufferLength: 2,
  },
}
```
