Next.jsは、プロジェクト内にTypeScriptエラーがある場合、**本番ビルド** (`next build`) を失敗させます。

アプリケーションにエラーがある場合でもNext.jsに危険を承知で本番コードを生成させたい場合、組み込みの型チェックステップを無効にすることができます。

このオプションを無効にする場合、ビルドやデプロイプロセスの一環として型チェックを実行していることを確認してください。そうしないと、非常に危険です。

`next.config.js`を開き、`typescript`の設定で`ignoreBuildErrors`オプションを有効にしてください：

```js title="next.config.js"
module.exports = {
  typescript: {
    // !! 警告 !!
    // プロジェクトに型エラーがある場合でも本番ビルドを成功させることを
    // 危険を承知で許可します。
    // !! 警告 !!
    ignoreBuildErrors: true,
  },
}
```
