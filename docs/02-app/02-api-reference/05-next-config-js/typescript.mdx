Next.js はプロジェクトに TypeScript のエラーがある場合、**本番ビルド**（`next build`）を失敗させます。

アプリケーションにエラーがある場合でも危険を承知で Next.js に本番コードを生成させたい場合は、組み込みの型チェックステップを無効にすることができます。

無効にする場合は、ビルドやデプロイのプロセスの一部として型チェックを実行していることを確認してください。そうしないと非常に危険です。

`next.config.js` を開き、`typescript` 設定で `ignoreBuildErrors` オプションを有効にします：

```js title="next.config.js"
module.exports = {
  typescript: {
    // !! WARN !!
    // プロジェクトに型エラーがある場合でも本番ビルドを成功させることを
    // 危険を承知で許可します。
    // !! WARN !!
    ignoreBuildErrors: true,
  },
}
```
