Next.js はデフォルトで `x-powered-by` ヘッダーを追加します。これをオプトアウトするには、`next.config.js` を開いて `poweredByHeader` の設定を無効にします：

```js title="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
