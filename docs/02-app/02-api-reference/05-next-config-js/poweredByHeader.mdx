Next.jsはデフォルトで`x-powered-by`ヘッダを追加します。これをオプトアウトするには、`next.config.js`を開いて`poweredByHeader`の設定を無効にしてください：

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
