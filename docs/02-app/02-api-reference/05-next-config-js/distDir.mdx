カスタムビルドディレクトリとして`.next`の代わりに使用するディレクトリ名を指定できます。

`next.config.js`を開き、`distDir`設定を追加します:

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

ここで`next build`を実行すると、Next.jsはデフォルトの`.next`フォルダの代わりに`build`を使用します。

> `distDir`はプロジェクトディレクトリから出ては**いけません**。たとえば、`../build`は**無効な**ディレクトリです。
