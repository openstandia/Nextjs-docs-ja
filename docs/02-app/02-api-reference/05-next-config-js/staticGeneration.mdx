`staticGeneration*`オプションでは、高度な使用ケースに対して静的生成プロセスを設定できます。

<Tabs>
<TabItem value="ts" label="TypeScript">

```ts title="next.config.ts" switcher
import type { NextConfig } from 'next'

// nextConfig変数を宣言し、静的生成の設定を含むオプションを指定
const nextConfig: NextConfig = {
  experimental: {
    // 失敗したページの生成を再試行する回数
    staticGenerationRetryCount: 1,
    // 各ワーカーが処理するページの最大数
    staticGenerationMaxConcurrency: 8,
    // 新しいワーカーを開始する前に処理するページの最小数
    staticGenerationMinPagesPerWorker: 25,
  },
}

// nextConfigオブジェクトをエクスポート
export default nextConfig
```

</TabItem>

## 設定オプション {#config-options}

利用可能なオプションは次のとおりです：

- `staticGenerationRetryCount`: 失敗したページの生成を再試行する回数
- `staticGenerationMaxConcurrency`: 各ワーカーが処理するページの最大数
- `staticGenerationMinPagesPerWorker`: 新しいワーカーを開始する前に処理するページの最小数
