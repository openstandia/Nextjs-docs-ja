---
title: 'serverExternalPackages'
description: '特定の依存関係を Server Components バンドルから除外し、ネイティブの Node.js `require` を使用します。'
---

[Server Components](/docs/app/building-your-application/rendering/server-components) と [Route Handlers](/docs/app/building-your-application/routing/route-handlers) の内部で使用される依存関係は、Next.js によって自動的にバンドルされます。

依存関係が Node.js の特定の機能を使用している場合、特定の依存関係を Server Components のバンドルから除外し、ネイティブの Node.js `require` を使用することを選択できます。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
