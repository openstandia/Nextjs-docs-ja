---
title: serverExternalPackages
description: サーバーコンポーネントのバンドルから特定の依存関係を除外し、ネイティブNode.jsの`require`を使用する
---

依存関係は[サーバーコンポーネント](/docs/app/building-your-application/rendering/server-components)および[ルートハンドラー](/docs/app/building-your-application/routing/route-handlers)内で使用されている場合、Next.jsによって自動的にバンドルされます。

依存関係がNode.js固有の機能を使用している場合は、特定の依存関係をサーバーコンポーネントのバンドルから除外し、ネイティブNode.jsの`require`を使用するように選択できます。

```js title="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
