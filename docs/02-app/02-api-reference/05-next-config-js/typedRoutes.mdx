---
title: 'typedRoutes'
description: '静的に型付けされたリンクの実験的サポートを有効にします。'
version: 'experimental'
---

[静的に型付けされたリンク](/docs/app/building-your-application/configuring/typescript#statically-typed-links) のための実験的サポートです。この機能を使用するには、プロジェクトでApp RouterおよびTypeScriptを使用する必要があります。

```js title="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true, // 静的に型付けされたリンクを有効にします
  },
}

module.exports = nextConfig
```
