---
title: 'reactMaxHeadersLength'
description: 'Reactによって出力され、レスポンスに追加されるヘッダーの最大長'
---

静的レンダリング中に、Reactはレスポンスに追加できるヘッダーを出力できます。これにより、フォント、スクリプト、スタイルシートなどのリソースをブラウザが事前にロードすることでパフォーマンスを向上させることができます。デフォルト値は`6000`ですが、`next.config.js`で`reactMaxHeadersLength`オプションを設定することでこの値を上書きできます：

```js title="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **Good to know**: このオプションはApp Routerでのみ利用可能です。

ブラウザとサーバー間のプロキシの種類によっては、ヘッダーが切り捨てられることがあります。たとえば、長いヘッダーをサポートしていないリバースプロキシを使用している場合、ヘッダーが切り捨てられないように、より低い値を設定する必要があります。
