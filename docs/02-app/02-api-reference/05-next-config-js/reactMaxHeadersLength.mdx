---
title: reactMaxHeadersLength
description: Reactによって生成され、レスポンスに追加されるヘッダーの最大長。
---

静的レンダリング中に、Reactはヘッダーを生成してレスポンスに追加することができます。これらはブラウザがフォント、スクリプト、スタイルシートなどのリソースを事前にロードすることを可能にすることで、パフォーマンスを向上させるのに役立ちます。デフォルト値は`6000`ですが、`next.config.js`の中で`reactMaxHeadersLength`オプションを構成することで、この値を上書きできます：

```js title="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **Good to know**: このオプションはApp Routerでのみ利用可能です。

ブラウザとサーバーの間にあるプロキシの種類によっては、ヘッダーが切り捨てられることがあります。例えば、長いヘッダーをサポートしていないリバースプロキシを使用している場合、ヘッダーが切り捨てられないようにするために、より低い値に設定する必要があります。
