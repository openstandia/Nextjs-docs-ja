---
title: 'reactMaxHeadersLength'
description: 'Reactによって生成され、レスポンスに追加されるヘッダーの最大長。'
---

静的レンダリング中に、Reactはヘッダーを生成し、レスポンスに追加することができます。これにより、フォント、スクリプト、スタイルシートのようなリソースをブラウザが事前に読み込むことを許可することで、パフォーマンスを向上させることができます。デフォルト値は`6000`ですが、`next.config.js`で`reactMaxHeadersLength`オプションを設定することでこの値を上書きできます：

```js title="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **Good to know**: このオプションはApp Routerでのみ使用できます。

ブラウザとサーバーの間のプロキシの種類によっては、ヘッダーが切り捨てられることがあります。たとえば、長いヘッダーをサポートしていないリバースプロキシを使用している場合、ヘッダーが切り捨てられないように、より低い値を設定する必要があります。
