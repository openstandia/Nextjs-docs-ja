---

`ImageResponse`コンストラクターを使用すると、JSXとCSSを用いて動的な画像を生成できます。これは、Open Graph画像やTwitterカードなどのソーシャルメディア用の画像を生成する際に便利です。

`ImageResponse`には以下のオプションがあります：

```jsx
import { ImageResponse } from 'next/og'

new ImageResponse(
  element: ReactElement,
  options: {
    width?: number = 1200
    height?: number = 630
    emoji?: 'twemoji' | 'blobmoji' | 'noto' | 'openmoji' = 'twemoji',
    fonts?: {
      name: string,
      data: ArrayBuffer,
      weight: number,
      style: 'normal' | 'italic'
    }[]
    debug?: boolean = false

    // HTTPレスポンスに渡されるオプション
    status?: number = 200
    statusText?: string
    headers?: Record<string, string>
  },
)
```

## 対応するCSSプロパティ {#supported-css-properties}

対応するHTMLとCSSの機能一覧については、[Satoriのドキュメント](https://github.com/vercel/satori#css)を参照してください。
