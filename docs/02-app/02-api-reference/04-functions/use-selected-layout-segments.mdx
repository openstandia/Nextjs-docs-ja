`useSelectedLayoutSegments`は**クライアントコンポーネント**フックで、呼び出し元のレイアウト**以下**のアクティブなルートセグメントを読むことができます。

これはパンくずリストなど、アクティブな子セグメントの知識が必要な親レイアウト内でのUIを作成するのに便利です。

```tsx filename="app/example-client-component.tsx" switcher
'use client'

import { useSelectedLayoutSegments } from 'next/navigation'

export default function ExampleClientComponent() {
  const segments = useSelectedLayoutSegments()

  return (
    <ul>
      {segments.map((segment, index) => (
        <li key={index}>{segment}</li>
      ))}
    </ul>
  )
}
```

```jsx filename="app/example-client-component.js" switcher
'use client'

import { useSelectedLayoutSegments } from 'next/navigation'

export default function ExampleClientComponent() {
  const segments = useSelectedLayoutSegments()

  return (
    <ul>
      {segments.map((segment, index) => (
        <li key={index}>{segment}</li>
      ))}
    </ul>
  )
}
```

> **Good to know**:
>
> - `useSelectedLayoutSegments`は[クライアントコンポーネント](/docs/app/building-your-application/rendering/client-components)フックで、レイアウトはデフォルトで[サーバーコンポーネント](/docs/app/building-your-application/rendering/server-components)であるため、通常はレイアウトにインポートされたクライアントコンポーネントを通じて呼び出されます。
> - 返されるセグメントには[ルートグループ](/docs/app/building-your-application/routing/route-groups)が含まれており、それらをUIに含めたくない場合があります。配列の`filter()`メソッドを使用して、角括弧で始まる項目を削除できます。

## パラメータ

```tsx
const segments = useSelectedLayoutSegments(parallelRoutesKey?: string)
```

`useSelectedLayoutSegments`は、_オプションで_ [`parallelRoutesKey`](/docs/app/building-your-application/routing/parallel-routes#useselectedlayoutsegments)を受け取り、そのスロット内でアクティブなルートセグメントを読み取ることができます。

## 戻り値

`useSelectedLayoutSegments`は、フックが呼び出されたレイアウトの一階層下のアクティブなセグメントを含む文字列の配列を返します。存在しない場合は空の配列を返します。

例えば、以下のレイアウトとURLの場合、返されるセグメントは次のようになります：

| レイアウト                  | 訪問したURL             | 返されるセグメント          |
| ------------------------- | --------------------- | --------------------------- |
| `app/layout.js`           | `/`                   | `[]`                        |
| `app/layout.js`           | `/dashboard`          | `['dashboard']`             |
| `app/layout.js`           | `/dashboard/settings` | `['dashboard', 'settings']` |
| `app/dashboard/layout.js` | `/dashboard`          | `[]`                        |
| `app/dashboard/layout.js` | `/dashboard/settings` | `['settings']`              |
