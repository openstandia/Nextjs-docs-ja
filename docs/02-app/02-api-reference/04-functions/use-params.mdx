`useParams`は、現在のURLによって埋め込まれているルートの[ダイナミックパラメータ](/docs/app/building-your-application/routing/dynamic-routes)を読み取るための**クライアントコンポーネント**フックです。

<Tabs>
<TabItem value="tsx" label="TypeScript">

```tsx title="app/example-client-component.tsx" switcher
'use client'

import { useParams } from 'next/navigation'

export default function ExampleClientComponent() {
  const params = useParams<{ tag: string; item: string }>()

  // ルート -> /shop/[tag]/[item]
  // URL -> /shop/shoes/nike-air-max-97
  // `params` -> { tag: 'shoes', item: 'nike-air-max-97' }
  console.log(params)

  return '...'
}
```
</TabItem>

## Parameters

```tsx
const params = useParams()
```

`useParams`は、パラメーターを受け取りません。

## Returns

`useParams`は、現在のルートの埋め込まれている[ダイナミックパラメータ](/docs/app/building-your-application/routing/dynamic-routes)を含むオブジェクトを返します。

- オブジェクト内の各プロパティは、アクティブなダイナミックセグメントです
- プロパティの名前はセグメントの名前で、プロパティの値はそのセグメントに埋め込まれている値です
- プロパティの値は、[ダイナミックセグメントのタイプ](/docs/app/building-your-application/routing/dynamic-routes)に応じて、`string`または`string`の配列になります
- ルートにダイナミックパラメータがない場合、`useParams`は空のオブジェクトを返します
- Pages Routerで使用する場合、`useParams`は初期レンダリングで`null`を返し、ルーターが準備できると上記のルールに従ってプロパティが更新されます

例えば：

| ルート                           | URL         | `useParams()`             |
| ------------------------------- | ----------- | ------------------------- |
| `app/shop/page.js`              | `/shop`     | `{}`                      |
| `app/shop/[slug]/page.js`       | `/shop/1`   | `{ slug: '1' }`           |
| `app/shop/[tag]/[item]/page.js` | `/shop/1/2` | `{ tag: '1', item: '2' }` |
| `app/shop/[...slug]/page.js`    | `/shop/1/2` | `{ slug: ['1', '2'] }`    |
