NextRequest は、追加の便利なメソッドを備えた [Web Request API](https://developer.mozilla.org/docs/Web/API/Request) を拡張しています。

## `cookies`

リクエストの [`Set-Cookie`](https://developer.mozilla.org/docs/Web/HTTP/Headers/Set-Cookie) ヘッダーを読み取るか、変更します。

### `set(name, value)`

名前を指定し、リクエストに対して指定された値のcookieを設定します。

```ts
// 着信リクエスト /home
// バナーを隠すためにcookieを設定します
// リクエストには `Set-Cookie:show-banner=false;path=/home` ヘッダーが含まれます
request.cookies.set('show-banner', 'false')
```

### `get(name)`

cookieの名前を指定し、そのcookieの値を返します。cookieが見つからない場合は `undefined` を返します。複数のcookieが見つかった場合は、最初のものを返します。

```ts
// 着信リクエスト /home
// { name: 'show-banner', value: 'false', Path: '/home' }
request.cookies.get('show-banner')
```

### `getAll()`

cookieの名前を指定して、そのcookieの値を返します。名前が指定されていない場合、リクエスト上のすべてのcookieを返します。

```ts
// 着信リクエスト /home
// [
//   { name: 'experiments', value: 'new-pricing-page', Path: '/home' },
//   { name: 'experiments', value: 'winter-launch', Path: '/home' },
// ]
request.cookies.getAll('experiments')
// あるいは、リクエストのすべてのcookieを取得します
request.cookies.getAll()
```

### `delete(name)`

cookieの名前を指定して、リクエストからcookieを削除します。

```ts
// 削除された場合は true を、削除されなかった場合は false を返します
request.cookies.delete('experiments')
```

### `has(name)`

cookieの名前を指定し、リクエスト上にそのcookieが存在する場合は `true` を返します。

```ts
// cookieが存在する場合は true を、存在しない場合は false を返します
request.cookies.has('experiments')
```

### `clear()`

リクエストから `Set-Cookie` ヘッダーを削除します。

```ts
request.cookies.clear()
```

## `nextUrl`

Next.js 固有のプロパティを含む、追加の便利なメソッドでネイティブの [`URL`](https://developer.mozilla.org/docs/Web/API/URL) API を拡張します。

```ts
// /homeへのリクエストを受けた場合、pathname は /home になります
request.nextUrl.pathname
// /home?name=lee へのリクエストを受けた場合、searchParams は { 'name': 'lee' } になります
request.nextUrl.searchParams
```

以下のオプションがあります：

<PagesOnly>

| プロパティ          | タイプ                     | 説明                                                                                                                               |
| ------------------- | ------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `basePath`          | `string`                  | URL の [ベースパス](https://nextjs.org/docs/canary/pages/api-reference/next-config-js/basePath) です。                                                    |
| `buildId`           | `string` \| `undefined`  | Next.js アプリケーションのビルド識別子です。[カスタマイズ](https://nextjs.org/docs/canary/pages/api-reference/next-config-js/generateBuildId)できます。  |
| `defaultLocale`     | `string` \| `undefined`  | [国際化](https://nextjs.org/docs/canary/pages/building-your-application/routing/internationalization) のデフォルトロケールです。                           |
| `domainLocale`      |                           |                                                                                                                                   |
| - `defaultLocale`   | `string`                  | ドメイン内のデフォルトロケールです。                                                                                               |
| - `domain`          | `string`                  | 特定のロケールに関連付けられたドメイン。                                                                                           |
| - `http`            | `boolean` \| `undefined` | ドメインが HTTP を使用しているかを示します。                                                                                       |
| `locales`           | `string[]` \| `undefined`| 利用可能なロケールの配列です。                                                                                                     |
| `locale`            | `string` \| `undefined`  | 現在有効なロケール。                                                                                                               |
| `url`               | `URL`                     | URL オブジェクト。                                                                                                                |

</PagesOnly>

<AppOnly>

| プロパティ       | タイプ                   | 説明                                                                                                                               |
| ---------------- | ----------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `basePath`       | `string`                | URL の [ベースパス](/docs/app/api-reference/next-config-js/basePath) です。                                                       |
| `buildId`        | `string` \| `undefined`| Next.js アプリケーションのビルド識別子です。[カスタマイズ](/docs/app/api-reference/next-config-js/generateBuildId)できます。     |
| `pathname`       | `string`                | URL のパス名です。                                                                                                                |
| `searchParams`   | `Object`                | URL の検索パラメータです。                                                                                                         |

> **Note:** Pages Router での国際化プロパティは App Router では使用できません。App Router を使用した [国際化についてさらに学ぶ](/docs/app/building-your-application/routing/internationalization)ことができます。

</AppOnly>
